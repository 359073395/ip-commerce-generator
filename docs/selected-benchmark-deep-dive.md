## 新增深挖方法卡

### 深挖方法：高咨询内容的证据链开头

- 方法：结果前置、金额/强对手、客户困境、过程难度、专业结果、咨询承接
- 场景：工程律师IP、高客单咨询、专业服务IP、B端服务
- 必填信息：结果或阶段性进展、金额/规模、强对手或复杂程度、客户真实困境、团队动作、电话/私信入口
- 输出骨架：第一句结果 -> 金额或强对手 -> 客户被压垮的处境 -> 团队怎么介入 -> 结果/进展 -> 电话咨询CTA
- 示例：工程律师内容不是先讲法条，而是先讲工程款到账、国央企强对手、老板快撑不住、团队风险共担。

### 深挖方法：风险共担的大白话成交表达

- 方法：不用专业名词、承认顾虑、风险共担、前期不收费、结果导向收费、电话承接
- 场景：律师服务、咨询服务、高客单服务、难成交服务
- 必填信息：客户不成交顾虑、收费方式、可承诺边界、服务结果、承接方式
- 输出骨架：说出顾虑 -> 解释为什么难 -> 大白话讲产品 -> 降低行动风险 -> 电话/私信CTA
- 示例：不要只说全风险代理，要说“你不是不想打，是现在被工程款卡住，前期不收费，要不回来分文不取”。

### 深挖方法：少评论高私信的沉默转化判断

- 方法：高隐私行业、评论少不等于无效、私信/电话优先、身份识别词、感同身受词
- 场景：工程律师IP、法律服务、债务回款、高隐私咨询、B端纠纷
