## 一句话结论

直播带货不是单靠主播喊卖，而是“人、货、场”共同完成成交。先选对品，再搭出让人信任和停留的直播间，用清晰的灯光、构图、场景、玩法和话术，把用户从进场、停留、点击商品到最终下单一步步推过去。

## 现在做直播带货晚不晚

课程开头先回答了一个常见担心：现在做直播带货是不是已经晚了。

课程判断：

```text
只要你开始做，就不算晚。
```

原因是头部主播减少或平台格局变化后，流量不会消失，而是会分散到更多中小直播间。对个人IP带货账号来说，机会仍然存在，只是要更精细地做内容、选品、场景和成交。

## 直播带货的三个要素

直播带货由三个核心元素组成：

```text
人 + 货 + 场
```

三者缺一不可。

- 人：主播形象、专业感、表达力、可信度。
- 货：商品是否有需求、利润、质量和适配度。
- 场：直播间场景、灯光、画面、氛围和购买理由。

课程举例：卖数码产品时，主播形象、桌面商品和背景设备要让用户一眼知道这是数码直播间；卖海参时，在港口或海货现场比在教室里卖更有信任感；小锅直播用极小的锅现场做蛋，靠稀缺感和好奇心把人留住。

## 三种常见带货形式

抖音上常见的带货形式包括：

- 直播带货。
- 视频带货。
- 图文带货。

直播带货像线上摊位，短视频带货像提前录好的销售展示，图文带货则更轻量，适合用图片和文字滚动展示商品卖点。

## 开通橱窗

如果想在抖音上卖货，需要先有自己的线上“摊位”，也就是商品橱窗。

大致路径：

```text
抖音
-> 我的
-> 右上角三条线
-> 抖音创作者中心
-> 全部
-> 电商带货
-> 成为带货达人
-> 实名认证
-> 缴纳保证金
```

课程中提到保证金约 500 元，后续通常可按规则退回，实际以平台页面为准。

## 商品来源

商品来源分两类：

### 1. 自有商品

适合：

- 自己有工厂。
- 自己有品牌。
- 自己有供应链。
- 自己有线下门店或产品资源。

如果卖自己的产品，需要开抖店，再把商品挂到橱窗里。

### 2. 非自有商品

适合还没有产品资源、想先做达人分销的人。

找货方法：

- 看同行卖什么。
- 在精选联盟找商品。
- 看热卖榜。
- 看热搜热度榜。
- 看搜索趋势。
- 用商品关键词搜索，比如猫粮、奶粉、健身器材等。

## 选品原则

好的带货不是随便挂商品，而是选适合账号和用户的品。

### 1. 先看利润

带货要赚钱，所以要看佣金和利润空间。

但不能只看高佣金，高佣金低质量会伤害粉丝信任。

### 2. 再看质量

看店铺评分、商品评分、物流体验和商家服务。

课程提醒：粉丝买到不好的货，第一反应往往是找主播，不是找上家。所以选品阶段要先把质量关。

### 3. 看是否适合自己赛道

美妆账号优先带美妆，健身账号优先带健身，宠物账号优先带宠物。

大众日用品可以适当跨赛道，但专业品类最好和账号定位匹配。

### 4. 看详情页是否装修好

商品详情页就像线下店面的装修。

要看：

- 商品介绍是否清楚。
- 多角度图片是否充分。
- 卖点是否完整。
- 背书信息是否有。
- 用户评价是否可靠。

详情页越清楚，用户点击商品后越容易下单。

## 什么是爆品

爆品不是单纯贵，也不是单纯奇怪，而是：

```text
当下有需求、大家想买、大家正在买，并且适合当前季节和赛道的商品。
```

例如家纺赛道，入夏前后更适合卖凉感、透气、降温相关商品，而不是厚被子。

选品要结合：

- 季节。
- 人群需求。
- 价格敏感度。
- 当前搜索趋势。
- 赛道消费习惯。

## 直播间场景类型

课程讲了几类常见直播场景。

### 1. 绿幕场景

适合：

- 团购。
- 大牌活动。
- 需要频繁换背景素材的直播。
- 需要展示套餐、图片、视频和价格信息的直播。

优点是灵活，能随时换主题，视觉信息丰富。

### 2. 工厂、仓库、原产地

适合：

- 源头工厂。
- 农产品。
- 茶叶。
- 服装。
- 食品。
- 有产地背书的商品。

价值是让用户觉得货多、源头、价格有优势。

### 3. 门店、柜台、货架

适合：

- 本地生活。
- 服装店。
- 运动品牌。
- 餐饮门店。
- 美妆集合店。

用户能看到真实门店和货架，更容易产生信任。

### 4. 室内主题场景

可以根据品类搭建：

- 厨房。
- 卧室。
- 美妆间。
- 客厅。
- 情感咨询空间。
- 教学空间。

核心是让场景和商品、账号定位一致。

### 5. 户外场景

适合：

- 旅游。
- 户外装备。
- 本地门店。
- 车展。
- 产地展示。

户外的真实感和环境冲击力强，但对设备、网络和收音要求更高。

## 直播间四种玩法

### 1. 获得感强

用户留下来，是因为能获得知识、方法或解决方案。

两种方式：

- 教一个方法。
- 解决当下问题。

例子：

- 教唱歌方法。
- 教健身动作。
- 教短视频脚本怎么写。
- 连麦解决情感、账号、选题问题。

适合知识型、技能型、咨询型个人IP。

可变现方向：

- 课程。
- 咨询。
- 工具。
- 相关商品。
- 服务套餐。

### 2. 效果很“牛”

核心是让用户亲眼看到产品效果。

玩法：

- 极限测试。
- 使用前后对比。
- 现场展示功能。
- 展示真实使用场景。

例子：

- 垃圾袋承重测试。
- 清洁剂半边擦拭对比。
- 冲锋衣防水展示。
- 美妆产品上脸前后对比。

用户眼睛看到效果，安全感和购买理由会更强。

### 3. 制作性强

核心是展示制作过程，让用户因为过程感、爽感、真实感停留。

适合：

- 手作。
- DIY。
- 食品制作。
- 街边小吃。
- 工厂加工。
- 玉石、饰品、玩具制作。

这种直播不一定一直讲卖点，过程本身就能吸引停留。

### 4. 正常带货

没有强知识、强效果、强制作过程时，就靠话术把商品价值讲清楚。

课程给了几个方向：

- 产品型：讲原料、工艺、配置、功能。
- 体验型：讲口感、触感、观感、使用感。
- 背书型：讲品牌、同款、联名、达人推荐。
- 比价型：讲市场价、同行价、直播间价、升级对比。

本质是让用户觉得：

```text
这个商品物有所值，甚至物超所值。
```

## 直播设备

主流直播方式：

- 手机直播。
- 电脑直播。

### 手机直播

优点：

- 成本低。
- 上手快。
- 一部手机就能开播。

不足：

- 画质和稳定性有限。
- 玩法少。
- 手机容易发热。
- 直播间质感不如电脑直播。

基础设备：

- 手机。
- 补光灯。
- 氛围灯。
- 麦克风。
- 手机支架。
- 手机散热器。

### 电脑直播

优点：

- 画质更好。
- 画面更稳定。
- 可以添加更多素材和组件。
- 更适合专业带货直播间。

基础逻辑：

```text
相机/摄像头采集画面
麦克风采集声音
采集卡汇总信号
电脑直播软件输出到抖音直播间
```

常见设备：

- 相机或摄像头。
- 镜头。
- 麦克风。
- 采集卡。
- 直播电脑。
- 抖音直播伴侣。
- 灯光系统。

## 直播灯光

课程讲了五类主要光源。

### 1. 主光

从人物斜前方打过来，是面部最主要的光源。

作用：

- 照亮主播。
- 决定整体亮度。
- 提升人物精神状态。

### 2. 辅光

从主光的另一侧补光，亮度弱于主光。

作用：

- 减少阴影。
- 让脸部更柔和。
- 避免画面过硬。

### 3. 顶光

从头顶或斜上方打下来。

作用：

- 补充头顶和肩部亮度。
- 让人物和背景更分离。

### 4. 轮廓光

从主播斜后方打过来。

作用：

- 勾出人物边缘。
- 提升立体感。
- 让主播和背景拉开层次。

### 5. 环境光和氛围光

用来照亮背景和营造直播间风格。

例如：

- 暖光营造温馨感。
- 冷光营造科技感。
- 小灯串营造情绪氛围。
- 背景投影增加层次。

## 直播构图

标准直播带货画面可以拆成三层：

```text
前景 + 中景 + 远景
```

### 前景

放商品和利益点。

可以展示：

- 包装。
- 数量。
- 颜色。
- 款式。
- 用料。
- 尺寸。
- 赠品。
- 使用前后对比。

### 中景

主播所在位置。

可以展示：

- 主播形象。
- 上身效果。
- 使用动作。
- 产品拿取。
- 讲解状态。

### 远景

背景和环境。

可以展示：

- 仓库货架。
- 门店氛围。
- 产地环境。
- 品牌背景。
- 活动主题。
- 风格化布景。

## 直播间画面信息区

直播画面里可以合理安排信息，不要浪费关键位置。

常见区域：

- 顶部：主题、福利、倒计时、活动信息。
- 侧边：产品细节、卖点图片、滚动素材。
- 底部：点击小黄车、下单引导、滚动字幕。
- 主播旁：主播身高体重、适用尺码、品牌背书等。

卖服装时，主播身高体重和试穿尺码尤其重要，因为用户会用主播做参照。

## 直播话术三大类

课程把直播话术分成三类：

```text
种草推荐话术
产品营销话术
直播数据话术
```

### 1. 种草推荐话术

目标：让用户产生“我想买”的欲望。

公式：

```text
切入需求
-> 提出问题
-> 给出解决方案
-> 塑造产品
-> 描绘结果/情绪价值
```

切入需求可以从：

- 特殊日期：生日、节日、纪念日。
- 使用场景：餐桌、通勤、旅行、办公室、学校、户外。
- 行为习惯：运动、护肤、做饭、带娃、熬夜。
- 关系人群：给老公、老婆、父母、孩子、闺蜜、朋友买。
- 痛点：吃、穿、住、行、用中的麻烦。

最后一定要把结果讲出来：

- 更好看。
- 更有面子。
- 更省事。
- 更舒服。
- 更显状态。
- 更有氛围。
- 更能解决问题。

### 2. 产品营销话术

目标：让用户觉得“我现在就应该在你这里买”。

可用方向：

- 产品溢价：品牌历史、设计灵感、原料、工艺、情绪价值。
- 人设背书：主播经验、专业身份、选品标准、真实使用。
- 价格锚点：市场价、原价、日常价、今天直播价。
- 促单理由：限时、限量、前多少名、赠品、错过没有。
- 售后保障：七天无理由、运费险、退换无忧、售后承诺。

报价不要乱报，要有层次：

```text
市场价/品牌价
-> 日常价
-> 今天直播间福利价
-> 限时限量促单
```

### 3. 直播数据话术

目标：提高直播间互动和平台推荐。

核心数据包括：

- 点赞。
- 评论。
- 关注。
- 灯牌/粉丝团。
- 进入率。
- 停留时长。
- 商品点击。
- 成交转化。

## 点赞引导

可以用三种方式：

- 目标激励：点赞到多少，上福利。
- 情绪共鸣：觉得我说得对的，点点赞。
- 趣味游戏：用小游戏或口令带动点赞。

重点是给用户一个点赞理由。

## 评论引导

可以用：

- 互动提问：有这个困扰的扣1。
- 需求二选一：想要A扣1，想要B扣2。
- 趣味参与：评论区扣想要的链接，扣得最多先上。

评论越多，直播间看起来越热闹，也更容易被平台识别为高互动。

## 关注引导

关注要给回报。

方式：

- 专属福利：关注后送优惠或赠品。
- 长期价值：每周固定分享选品、避坑、福利。
- 情感拉近：以后买这类东西先来我这里看。

不要只干喊“点关注”，要告诉用户关注你能得到什么。

## 提高直播间进入率

进入率取决于用户刷到直播间时，愿不愿意点进来。

提升方式：

- 场景足够清楚。
- 福利预告明确。
- 画面能让人一眼看懂卖什么。
- 话术直接给进场理由。

示例方向：

```text
直播间3分钟后开秒杀。
夏天怕晒黑的进来，今天教你一招防晒穿搭。
新人进来先领专属福利。
```

## 提高停留时长

用户停得越久，越有机会听到卖点和成交理由。

方法：

- 进来先别走，马上讲福利。
- 用倒计时、限时活动留人。
- 用痛点解决型内容留人。
- 用对比展示留人。
- 用福袋、小活动留人。

## 提高商品点击

商品点击是成交前的重要动作。

引导方式：

- 直接推荐：小黄车1号链接，真心推荐你们看。
- 故事共鸣：这款是我出差时被种草的。
- 互动背书：刚才很多人问的就是这个。
- 评价引导：你们可以去看商品下面的真实评价。

商品详情页要装修好，否则用户点进去也不容易下单。

## 提高成交转化

成交话术可以结合：

- 情感共鸣。
- 信任背书。
- 真实体验。
- 价格对比。
- 限时限量。
- 售后保障。

要提前替用户说出顾虑：

- 不合适能不能退？
- 质量不好怎么办？
- 尺码不合适怎么办？
- 会不会踩坑？
- 有没有运费险？

你先把顾虑解除，用户更容易下单。

## 直播间不能说什么

课程提醒直播违规词和高风险表达要提前规避。

常见风险：

- 涉及政治、国家机关、领导人等敏感内容。
- 极限词：最低、第一、顶级、最好的、全网最低等。
- 虚假宣传：夸大功效、没有资质却宣称功效。
- 医疗化、功效化表达：治疗、美白、瘦身等要有合规依据。
- 诱导关注或附加条件：关注才能发货、加团才能拍等。
- 私下引流：加微信、私信送、平台外交易等。

直播话术要提前过一遍，尤其是美妆、食品、保健、减肥、母婴等赛道。

## 直播心态

课程最后强调：直播难点不只在技术，也在心态。

第一场直播数据不好很正常，不要一开始就给自己设太高期待。

正确做法：

- 多播。
- 多复盘。
- 多改话术。
- 多调整场景。
- 多测试商品。
- 坚持到第30场、第100场再回看自己。

直播和短视频一样：

```text
多拍、多写、多剪、多复盘，总会越来越顺。
```

## 直播间搭建检查表

```text
1. 人、货、场是否一致？
2. 商品是否适合账号赛道？
3. 商品佣金和质量是否过关？
4. 店铺评分、物流分、服务分是否看过？
5. 详情页是否装修完整？
6. 直播场景是否一眼看懂卖什么？
7. 主播、商品、背景是否有层次？
8. 主光、辅光、顶光、轮廓光、环境光是否合理？
9. 前景是否展示了利益点和赠品？
10. 话术是否覆盖需求、卖点、价格、促单和售后？
11. 是否设计了点赞、评论、关注和商品点击引导？
12. 是否提前排查违规词和风险表达？
```

## 行动清单

- 先确定账号赛道，再选商品，不要乱挂无关品。
- 从精选联盟和同行账号里各找10个候选品。
- 每个候选品记录佣金、评分、销量、详情页、评价和适配度。
- 直播前写清人货场定位：我是谁、卖什么、在什么场景卖。
- 先用手机低成本试播，再根据数据决定是否升级电脑直播。
- 每场直播准备至少3组福利、3组互动口令、3组促单话术。
- 每场结束复盘进入率、停留、商品点击、成交和违规提醒。
- 连续播满30场后，再做一次直播间大调整。
