## 一句话结论

短视频不是拍得磕磕巴巴就废了，很多内容都可以靠剪辑救回来。剪辑小白先掌握剪映的基础功能，再按“粗剪、精剪、封面”的流程做，就能把普通素材剪成节奏清楚、字幕舒服、画面有重点的视频。

## 剪辑心态

课程强调：

```text
三分拍，七分剪。
```

拍摄时卡顿、说错、停顿、重复、忘词都很正常。关键是后期能不能把气口剪掉，把节奏拉起来，把用户该看的信息呈现清楚。

剪辑高手来自重复练习，不是看一遍教程就会。

## 工具：剪映

课程使用剪映作为核心剪辑工具，手机、平板、电脑都可以操作。

基础流程：

```text
打开剪映
-> 开始创作
-> 选择素材
-> 添加
-> 进入剪辑工程
-> 用底部工具栏完成剪辑
-> 导出
```

## 常用基础功能

### 1. 分割

把一段素材从指定位置切开。

用途：

- 删除说错部分。
- 删除停顿。
- 调整节奏。
- 拆分镜头。

### 2. 移动素材

长按素材，把它拖到前后不同位置。

用途：

- 调整镜头顺序。
- 把开头素材移到前面。
- 重组故事结构。

### 3. 变速

调整素材播放速度。

注意：

- 速度越小越慢。
- 速度越大越快。
- 口播视频不要乱用大幅变速，否则声音和口型会变形。

### 4. 音量

调整单段素材音量，也可以统一应用到全部素材。

注意：人声永远应该比背景音乐更清楚。

### 5. 动画

包括入场动画、出场动画、组合动画。

用途：

- 让画面或文字更有动态。
- 适合做转场、标题、重点信息。

不要滥用，太多会显得廉价。

### 6. 音频分离

把视频里的声音从画面中分离出来，变成独立音频轨道。

用途：

- 保留声音，替换画面。
- 单独调整人声。
- 做画面重组。

### 7. 编辑

常用功能：

- 镜像。
- 旋转。
- 裁剪。
- 调整比例。

短视频发布通常用 `9:16` 竖屏比例。

### 8. 滤镜与调节

可以调整画面风格、亮度、饱和度、对比度等。

原则：提高质感，不要把画面调得假。

### 9. 蒙版

可以让画面只显示某个形状范围，比如圆形、矩形等。

用途：

- 做分屏。
- 做局部展示。
- 做对比画面。

### 10. 透明度

控制素材清晰或透明程度。

用途：

- 叠加画面。
- 做淡化效果。
- 做氛围层。

### 11. 复制、倒放、定格

复制：快速多出一份素材。

倒放：让动作反向播放。

定格：让某一帧静止，适合强调表情、结果、关键瞬间。

### 12. 文本与智能字幕

文本可手动输入；智能字幕可以自动识别口播内容生成字幕。

智能字幕生成后必须人工检查错字、漏字和断句。

### 13. 贴纸、画中画、特效

贴纸：增加提示、情绪或趣味元素。

画中画：在原有视频上再叠加一个视频或图片。

特效：分为画面特效和人物特效。

原则：只在帮助理解或增强重点时使用，不要堆效果。

### 14. 导出

普通短视频直接用默认推荐参数即可。平台会压缩视频，普通手机素材不必过度追求高参数。

## 剪辑流程

### 第一步：粗剪

粗剪的目标是让视频能顺畅看完。

流程：

```text
导入素材
-> 调整音量
-> 开启降噪
-> 添加智能字幕
-> 剪气口
-> 删除说错、重复、停顿
```

### 第二步：精剪

精剪的目标是让视频更好看、更有重点。

可加：

- 画中画。
- 蒙版。
- 贴纸。
- 关键帧。
- 特效。
- 音乐。
- 重点字变色。
- 封面设计。

## 剪气口

剪气口是剪辑里最基础也最重要的能力。

要剪掉：

- 长停顿。
- 重复说的话。
- 说错的话。
- 卡壳。
- 等待。
- 无意义空白。

剪气口的价值：

- 提升节奏。
- 减少拖沓。
- 增强完播。
- 让口播更像一气呵成。

## 剪气口的两个常见错误

### 1. 气口留太长

两个镜头之间停顿太久，会让视频卡一下，节奏掉下来。

### 2. 吞字

前一个字还没说完，后一个镜头就接上了，用户会听不清。

正确目标：

```text
字接字，但不吞字。
```

## 字幕规范

### 1. 不要超出安全边

平台发布后会压缩和叠加标题、昵称、按钮。字幕不要放太靠边，否则会被裁掉或挡住。

### 2. 不要两行大段字幕

短视频用户是来放松的，不是来读论文的。字幕太长、两三行堆在一起，会增加阅读压力。

做法：

- 拆短句。
- 每屏少字。
- 让字幕跟说话节奏走。

### 3. 字体不要用默认系统字体

课程建议使用更好看的字体，例如新青年体、出成体等。重点是字体要清楚、舒服、有辨识度。

### 4. 字幕位置

常用位置：

- 下方三分之一偏上一点。
- 上方三分之一偏下一点。

一般更推荐放下方，但不要太低，避免被平台文案和按钮遮挡。

### 5. 检查错字

智能字幕一定会有识别错误，发布前必须检查。

### 6. 重点字可变色

重点字可以换色，但颜色不要超过三种。颜色太多会显得乱。

## 关键帧

关键帧可以让素材在两个时间点之间产生动画变化。

简单理解：

```text
第一个点记录当前状态
第二个点记录变化后的状态
中间自动生成动画
```

可以改变：

- 大小。
- 位置。
- 颜色。
- 透明度。
- 字幕位置。

常见用法：

- 画面慢慢放大。
- 字幕从下往上滚动。
- 颜色逐渐变化。
- 图片从一侧移动到另一侧。

关键帧距离越近，变化越快；距离越远，变化越慢。

## 画中画与贴纸

画中画适合：

- 插入截图。
- 插入案例。
- 插入对比画面。
- 展示聊天记录。
- 展示产品细节。

注意：

- 位置要对。
- 大小要合适。
- 出现顺序要和讲述内容一致。

贴纸适合：

- 加提示。
- 加情绪。
- 加趣味。
- 强调重点。

不要为了“热闹”乱加贴纸。

## 蒙版

蒙版可以做局部画面显示，例如把两个画面框起来、做左右对比、做圆形头像区域。

常用调节：

- 形状。
- 长宽。
- 圆角。
- 羽化。
- 位置。

羽化可以让边缘更自然，不那么生硬。

## 封面为什么重要

封面有三个作用：

- 方便主页浏览。
- 提高点击率。
- 形成个人IP或系列统一风格。

一个账号如果每个系列封面风格统一，用户会更容易识别，也更容易觉得账号有价值。

## 封面常见错误

- 画面太暗。
- 字太多。
- 没有重点。
- 直接随机截一帧。
- 人物状态不好。
- 封面和内容关系不强。

## 封面制作思路

基础步骤：

```text
选择视频里好看的一帧
-> 复制该画面
-> 转成画中画
-> 添加标题文字
-> 用智能抠像把人物抠出来
-> 调整层级，让文字在人物后面或前后有层次
-> 截取为封面
```

这种方式可以做出“文字在人物后面”的立体感，比普通截图更有质感。

## 音乐与素材库

课程建议提前收藏音乐素材，并按用途分类：

- 高燃。
- 动感。
- 可爱。
- 浪漫。
- 舒缓。
- 情绪。
- 萌宠。
- 健身。
- 美妆。

剪辑时根据视频情绪选音乐，不要随便套热门音乐。

## 辅助工具

可用于封面和修图：

- 醒图：做封面、套模板、修图。
- 美图类工具：可适度修图。

课程提醒：拍摄时尽量不要过度美颜，保持原生真实；如果原相机实在接受不了，可以适度修，但不要修到不真实。

## 剪辑检查表

```text
1. 音量是否足够，人声是否清楚？
2. 是否开启降噪？
3. 气口是否剪干净？
4. 有没有吞字？
5. 字幕是否在安全区内？
6. 字幕是否太长或超过两行？
7. 字幕错字是否检查过？
8. 重点字颜色是否超过三种？
9. 画中画/贴纸/蒙版是否服务内容？
10. 封面是否有重点、够亮、能点击？
```

## 行动清单

- 先练分割、删除、移动、字幕、音量、导出。
- 每条口播都必须剪气口。
- 智能字幕生成后逐句检查。
- 先做粗剪，再做精剪。
- 每周模仿拆解3个封面。
- 建立自己的音乐素材收藏夹。
- 不追求花哨，先追求清楚、顺畅、舒服。
