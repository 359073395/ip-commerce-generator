## 一句话结论

观点脚本有了选题后，要用结构和论据把观点讲成立。常用结构是“归纳论证”和“正反对比”，常用论据包括权衡利弊、无奈之举、不同视角、适度攻击、倒打一耙和时间维度。开篇要把情绪拉高，否定句通常比肯定句和疑问句更能留人。

## 承接上一节

上一节的核心是：

```text
给他想做的事找个理由；
给他不想做或没做成的事找个借口。
```

并且学了两类观点选题：

- 人群类选题。
- 行业类选题。

这一节解决的是：

- 观点脚本怎么组织结构。
- 论据怎么找。
- 开篇怎么写得更有情绪。
- 如何用AI辅助生成脚本。

## 两种脚本结构

### 1. 归纳论证

结构：

```text
开篇提出一个争议事件/观点
-> 论据1
-> 论据2
-> 论据3
-> 总结观点
```

它类似“总分总”。

适合：

- 观点清楚。
- 想逐步说服用户。
- 有多个理由支撑。
- 适合做情绪铺垫。

### 2. 正反对比

结构：

```text
先讲一个反面例子
-> 再讲一个正面例子
-> 总结观点
```

适合：

- 需要对比两种选择。
- 需要突出差异。
- 需要让用户看到后果。
- 需要把观点讲得更鲜明。

例子：

```text
反面：她一直忍，最后所有人都觉得她好欺负。
正面：她开始给自己设边界，关系反而变舒服了。
观点：善良不是没有边界，真正的温柔要先保护自己。
```

## 六类论据

课程给了六种论据类型，用来支撑观点。

### 1. 权衡利弊

核心问题：

```text
这么做会得到什么？
不这么做会失去什么？
这么做的好处是什么？
不这么做的坏处是什么？
```

适合解释选择题。

例子：

```text
女人要不要保持经济独立？
不独立，你失去的是选择权；
独立，你得到的是遇到问题时转身的底气。
```

### 2. 无奈之举

核心话术：

```text
我也懂。
我也知道。
但真的没有办法。
```

它的作用是替用户找理由，让用户觉得被理解。

例子：

```text
我也知道全职妈妈很辛苦，也知道很多人想陪孩子长大。
但现实是孩子奶粉钱要花，学费要花，生活每一笔都要花。
不是我们不想停下来，是压力不允许。
```

这类论据特别适合安慰用户。

### 3. 不同视角

同一件事可以从不同角度说。

常见视角：

- 当事人怎么看。
- 旁观者怎么看。
- 家人怎么看。
- 伴侣怎么看。
- 孩子怎么看。
- 消费者怎么看。
- 从业者怎么看。

注意：不同视角都要服务同一个观点，不要互相打架。

例子：

```text
女人花钱变美有没有错？
站在自己的角度，是取悦自己；
站在伴侣的角度，他带你出去也有面子；
站在孩子的角度，他也会为有一个精神漂亮的妈妈开心。
```

### 4. 适度攻击

课程里称为“人生攻击”，但更安全的理解是：对某类抽象对象进行情绪化批判。

不建议随便攻击真实个人。

可攻击对象：

- 所谓专家。
- 行业乱象。
- 虚假成功学。
- 不合理规则。
- 消费骗局。

适合人设很强、表达有冲击力的账号。

风险：

- 容易显得刻薄。
- 容易引战。
- 容易偏离目标用户。

普通个人IP带货账号慎用。

### 5. 倒打一耙

倒打一耙不是胡搅蛮缠，而是把用户的委屈转成“不是我的错，是另一个原因导致的”。

常见句式：

```text
这不是你不努力，是环境给你的回报太少。
不是你不会过日子，是这几年生活成本涨得太快。
不是你不想变好，是你一直没人帮你把路讲清楚。
```

它能快速提供情绪价值。

但要注意不要变成无底线甩锅，观点仍然要自洽。

### 6. 时间维度

从时间线上看问题，让观点更深刻。

可用角度：

- 从过去看现在。
- 从现在看未来三年。
- 从现在看未来十年。
- 从历史变化看当下选择。
- 从社会趋势看个人处境。

例子：

```text
现在投资自己，三年后你会感谢今天咬牙坚持的自己。
```

时间维度更考验洞察力。如果暂时写不出来，可以先用生活小故事支撑观点。

## 论据不需要全用

课程强调：六类论据不是每条视频都要全部用。

正确用法：

```text
挑1-2个最适合这个观点的论据，讲透。
```

如果AI或素材库给了很多论据，只选最有情绪、最适合自己表达的。

## AI辅助写观点脚本

可以把以下信息交给AI：

- 选题。
- 目标用户。
- 观点立场。
- 脚本结构。
- 六类论据。
- 只要文案，不要画面。

但AI只是辅助。

课程提醒：

```text
AI的价值是给你内容；
人的价值是判断内容好坏。
```

你要判断：

- 哪个论据适合你。
- 哪句话像真人说话。
- 哪个点有情绪。
- 哪些内容需要删掉。
- 哪些需要换成自己的经历。

## 开篇怎么更抓人

观点脚本的开篇要让情绪更强。

课程给了一个重要排序：

```text
否定句 > 肯定句 > 疑问句
```

疑问句没有答案，情绪弱。

肯定句有立场，情绪更强。

否定句通常带冲突，情绪最强。

例子：

弱：

```text
结婚后过年一定要去男方家吗？
```

强：

```text
结婚后过年，不是必须回男方家。
```

更强：

```text
结婚后过年还默认去男方家，本来就不公平。
```

观点内容里，用户更容易被明确立场留住。

## 开篇情绪要递增

课程用多个例子说明：同一个话题，越有情绪、越有立场，越容易留人。

比如“应试教育和兴趣教育”：

弱：

```text
应试教育和兴趣教育应该如何平衡？
```

强：

```text
建议学校不要再只看考试成绩。
```

更强：

```text
学校的考试成绩，就应该取消排名。
```

不是说每个观点都要极端，而是开篇要给出足够明确的态度。

## 观点脚本完整路径

```text
1. 确定写作对象。
2. 选择人群类或行业类选题。
3. 找人物关系或行业对象。
4. 找冲突事件。
5. 确定立场。
6. 优化开篇，让情绪更强。
7. 选择脚本结构：归纳论证或正反对比。
8. 选择1-2类论据。
9. 用自己的语言写成口播。
10. 结尾落到一句清晰观点。
```

## 行业类观点脚本例子

选题：

```text
20万买新车，还是买原价40万的二手车？
```

立场：买新车更稳。

可用论据：

- 权衡利弊：二手车维修保养风险更高。
- 无奈之举：我也想买看起来更有面子的二手豪车，但我不会判断车况。
- 不同视角：别人会觉得你为了面子买车，而不是为了实用。
- 时间维度：未来几年新车保值、维修和用车成本更可控。

脚本方向：

```text
20万预算，普通人别硬上二手豪车。
不是豪车不好，是你承担不起它后面的不确定。
```

## 人群类观点脚本例子

选题：

```text
女人该不该做全职妈妈？
```

立场：不要轻易失去经济独立。

可用论据：

- 无奈之举：我也知道陪孩子重要，但生活每一笔钱都要有人承担。
- 权衡利弊：经济独立不只是赚钱，是保留选择权。
- 时间维度：几年后重返职场，成本会比现在更高。

脚本方向：

```text
女人不是不能做全职妈妈，而是不能在没有退路的时候，把自己彻底交出去。
```

## 个人IP带货怎么用

观点脚本可以先吸引同频用户，再自然承接产品或服务。

例子：

美妆带货：

```text
女人化妆不是为了取悦谁，是为了在镜子里先看见一个愿意照顾自己的自己。
```

家居带货：

```text
家务不是女人的天职，一个家舒服不舒服，从来不是一个人的责任。
```

母婴带货：

```text
宝妈不是矫情，她只是太久没有被当成一个需要被照顾的人。
```

产品不要硬插，要先让用户认可你的观点，再把产品作为解决方案或生活场景自然出现。

## 常见错误

- 开篇只提问，没有立场。
- 论据和观点打架。
- 想把六类论据全塞进去。
- 只讲道理，没有情绪。
- 站错目标用户。
- 把粉丝当客户而不是朋友。
- AI生成后不判断，直接照读。
- 用过度攻击制造冲突。

## 聊观点脚本模板

```text
[明确立场的开篇]

很多人觉得[常见看法]，
但我反而觉得[你的观点]。

先说第一点，[论据1]。
[用故事/细节/现实场景解释]

再说第二点，[论据2]。
[从另一个角度补充]

所以这件事不是[错误理解]，
而是[你的核心判断]。

[一句金句式总结]
```

正反结构模板：

```text
[明确立场的开篇]

我见过一种人，[反面例子]。
他以为自己是在[表面行为]，
其实是在[真实问题]。

但另一种人会[正面做法]，
因为他知道[关键认知]。

所以[总结观点]。
```

## 检查表

```text
1. 开篇是不是明确立场？
2. 开篇是不是比疑问句更有情绪？
3. 论据是不是都在支撑同一个观点？
4. 有没有选1-2个论据讲透？
5. 有没有用无奈、利弊、视角或时间维度提供情绪？
6. 有没有避免和目标用户对抗？
7. 结尾有没有落到一句清晰观点？
8. 这条内容能不能让用户觉得“他懂我”？
```
