## 一句话结论

普通选题通常只能做到60分，想让它更容易被点击和传播，要在原选题上叠加爆款元素。爆款元素不是玄学，而是用户天然关心的东西：人群、成本、极端、反差、怀旧、名人/富豪、内幕、猎奇等。选题公式可以理解为：基础方向 + 爆款元素 + 目标人群。

## 普通选题为什么不容易爆

课程把普通选题比作一盘普通牛排：

```text
能吃，但卖不上高价。
```

普通选题比如：

- 如何快速收拾房间。
- 每周复盘要注意什么。
- 教你如何点外卖不踩雷。
- 数码好物分享。

这些选题能拍，但很难天然爆，因为它们只是“有用”，不一定“想点”。

要把普通选题变成爆款选题，需要加“爆款元素”。

## 选题升级公式

```text
普通选题 + 爆款元素 = 更有点击欲的选题
```

例子：

```text
如何点外卖不踩雷
-> 挑战吃差评第一的外卖
```

```text
买车应该注意什么
-> 20万预算，什么车最适合高分女生
```

```text
经典歌曲盘点
-> 1996年的十大金曲
```

升级后，用户不只是觉得“有用”，还会有好奇、代入、比较、情绪或回忆。

## 爆款元素1：人群

给选题加上明确人群，可以让目标用户觉得“这条和我有关”。

常见人群：

- 女生。
- 男生。
- 宝妈。
- 老板。
- 上班族。
- 高分女生。
- 新手小白。
- 外行人。
- 穷人/普通人。
- 有钱人/富豪。

例子：

```text
什么车值得买
-> 20万预算，高分女生适合买什么车
```

```text
怎么开店
-> 没开过店的人，第一次开店要避开的坑
```

个人IP带货启发：先明确你要吸引谁，再给这个人群定制选题。

## 爆款元素2：成本

用户天然关心钱、时间、精力成本。

常见表达：

- 花多少钱。
- 不花钱。
- 用一周。
- 3个月。
- 100元预算。
- 只用10分钟。
- 一天能不能完成。

例子：

```text
如何整理衣柜
-> 不花钱，3小时整理出一个能多放50件衣服的衣柜
```

```text
怎么做选品
-> 100元预算，怎么测试一个带货品能不能卖
```

成本越具体，用户越容易判断值不值得看。

## 爆款元素3：极端

极端包括最高、最低、最差、最快、最贵、最便宜、第一、倒数第一等。

它能制造好奇：

```text
到底有多好？
到底有多差？
为什么是第一？
为什么是倒数第一？
```

例子：

```text
外卖测评
-> 挑战吃评分最低的外卖
```

```text
美甲店体验
-> 去全城差评最多的美甲店做一次美甲
```

使用时要注意真实体验和表达边界，避免恶意攻击或虚假评价。

## 爆款元素4：反差

反差是把不该放在一起的东西放在一起。

常见反差：

- 老年人做年轻人的事。
- 富人做普通人的事。
- 穷人体验高端消费。
- 高端品牌做低价产品。
- 看起来不靠谱的人做专业事。
- 严肃场景出现轻松行为。

例子：

```text
老年人去年轻人最爱的店，会发生什么？
```

```text
让外行人去挑沙发，最容易踩哪些坑？
```

反差会让用户停下来，因为大脑想知道结果。

## 爆款元素5：怀旧

怀旧会调动用户的记忆和情绪。

常见表达：

- 1996年的十大金曲。
- 童年零食。
- 小时候的夏天。
- 以前的人怎么生活。
- 90后/00后的共同回忆。

适合：

- 情绪内容。
- 音乐。
- 食品。
- 家居。
- 穿搭。
- 个人故事。
- 老物件/老品牌。

怀旧不是单纯说过去，而是唤起用户“我也经历过”的感觉。

## 爆款元素6：名人、富豪、大牌

用户天然对强势身份和资源感兴趣。

常见方向：

- 身家十亿的人怎么花钱。
- 明星同款。
- 大牌平替。
- 名人用过的东西。
- 富豪家庭怎么选择。
- 电视剧里出现的同款。

例子：

```text
身家十亿的老板，一天到底花多少钱？
```

```text
电视剧里出现的那款沙发，现实中值多少钱？
```

注意：涉及名人、大牌和同款时，要避免虚假背书和未经证实的宣传。

## 爆款元素7：内幕、黑幕、外行不知道

用户喜欢知道“别人不知道的信息”。

常见表达：

- 外行人不知道的坑。
- 行业内不会告诉你的秘密。
- 买沙发前必须知道的内幕。
- 某行业的黑心套路。
- 新手最容易被坑的地方。

例子：

```text
外行人不知道的5个沙发选购坑。
```

```text
第一次做小黄车带货，最容易亏钱的3个地方。
```

个人IP带货很适合用这个元素，因为它能建立专业感和信任感。

## 爆款元素8：猎奇/离谱/奇怪

课程中多次提到“有病”“奇怪”“离谱”这类表达，本质是猎奇元素。

常见方向：

- 奇怪但有用的产品。
- 离谱但真实的体验。
- 一般人想不到的玩法。
- 让人想验证真假的说法。

例子：

```text
这些看起来很离谱的厨房小工具，真的有用吗？
```

```text
我买了一个全网都说很奇怪的收纳神器。
```

猎奇要服务内容，不要为了怪而怪。

## 选题不是只垂直产品

课程后半段特别提醒：很多人把“垂直”理解得太窄，以为美妆账号只能讲美妆，茶账号只能讲茶。

更准确的理解是：

```text
垂直不是只垂直产品，而是垂直目标人群。
```

比如一个美妆账号，目标是年轻女性。年轻女性不只关心护肤美妆，也可能关心：

- 情感。
- 赚钱。
- 穿搭。
- 旅行。
- 生活方式。
- 职场。
- 消费避坑。

这些内容和产品不完全一样，但和目标人群相关。

## 内容配比：赛道为主，人群为辅

虽然可以扩展人群话题，但不能完全脱离账号主赛道。

正确做法：

```text
赛道内容为主
人群内容为辅
```

如果你是美妆账号，大部分内容仍然要围绕变美、护肤、妆容、产品体验；少量内容可以扩展到女性情绪、消费、穿搭和生活方式。

如果全部都拍，用户会不知道你到底是做什么的。

## 把爆款选题变成系列

课程给了一个重要方法：用“定量 + 变量”把爆款选题做成系列。

```text
定量：系列里不变的部分。
变量：每一集变化的部分。
```

例子：

```text
定量：改造
变量：不同房间/不同预算/不同户型
```

```text
定量：测评
变量：不同品牌/不同价格/不同场景
```

```text
定量：外行不知道的坑
变量：沙发、床垫、茶叶、护肤品、直播带货
```

系列的好处：

- 降低每次想选题的成本。
- 让用户有追更感。
- 方便账号形成栏目。
- 一条跑出来后可以持续复用结构。

## AI辅助选题

课程提到可以用AI辅助生成选题，但不要完全照搬。

更好的用法：

- 给AI一个普通选题，让它帮你改成更有点击欲的选题。
- 告诉AI你的行业、人群、账号定位，让它输出多个方向。
- 再由你判断哪些符合人设、平台和执行成本。

注意：

```text
AI给的是参考，不是最终答案。
```

AI生成的选题容易官方、空泛、像模板，需要人工改成人话。

## 爆款选题练习法

```text
1. 先写一个普通选题。
2. 给它加一个人群。
3. 再加一个爆款元素。
4. 判断是否和账号赛道相关。
5. 判断执行成本是否能承受。
6. 判断是否能做成系列。
```

示例：

```text
普通：如何挑选茶叶
加人群：送领导怎么挑茶叶
加元素：送领导茶叶，最容易踩的3个坑
系列化：送领导、送长辈、送客户、送老师分别怎么选茶
```

## 个人IP带货应用

个人IP带货账号可以这样改选题：

```text
普通：这款锅很好用
升级：100元以内，租房党最值得买的一口锅
```

```text
普通：怎么选护肤品
升级：外行人不知道的护肤品黑心套路
```

```text
普通：这个收纳盒很方便
升级：我买了全网最离谱的收纳盒，结果真香了
```

```text
普通：直播间怎么搭
升级：新手直播间最烧钱的3个无效布置
```

## 选题检查表

```text
1. 这个选题有没有明确人群？
2. 有没有叠加成本、极端、反差、怀旧、内幕等元素？
3. 用户看了标题会不会想知道结果？
4. 这个选题是否符合账号目标人群？
5. 是否仍然围绕主赛道，不会让账号失焦？
6. 执行成本是否可控？
7. 能不能拆成一个系列？
8. 是否避免虚假宣传、恶意评价和夸张承诺？
```

## 行动清单

- 把现有10个普通选题，各加一个爆款元素重写。
- 每个选题至少写3版：人群版、成本版、内幕版。
- 选一个表现最好的选题，拆出定量和变量，做成5集系列。
- 每周复盘哪些爆款元素在自己账号里更有效。
- AI生成选题后必须人工改写，确保像真人表达。
- 主赛道内容占大头，人群扩展内容做辅助。
