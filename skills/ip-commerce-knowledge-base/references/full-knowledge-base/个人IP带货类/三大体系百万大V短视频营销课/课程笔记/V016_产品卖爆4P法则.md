## 一句话结论

带货视频不能拍成生硬广告，而要像一个有温度、有灵魂的电商人在给朋友分享好东西。4P法则的核心是：先圈定目标人群，再引入场景痛点，接着推出产品卖点，最后给出促销行动。

## 带货前的账号节奏

课程提醒：前期不要账号一起来就全发带货视频，因为信任不足，卖不动，也容易让用户觉得你只是广告号。

建议：

- 先做人设和内容方向。
- 粉丝积累到一定基础后，再尝试人设带货。
- 内容比例可按实际情况调整，带货不要压过人设。
- 不要等到十几万、二十万粉丝才开始变现，变现要趁早测试。

核心心态：

```text
不是纯卖货，而是用人设带货。
```

## 4P法则

课程中的4P可整理为：

```text
People：目标人群
Pain/Problem：场景痛点
Product：产品卖点
Promotion：促销行动
```

### 1. People：圈定目标人群

开头先让目标用户知道“这条视频和我有关”。

常见人群：

- 宝妈。
- 健身党。
- 懒人。
- 上班族。
- 学生党。
- 化妆新手。
- 正在减肥的人。
- 经常做饭的人。
- 需要送礼的人。

开头要直接点名或用场景暗示。

```text
宝宝不爱洗澡的宝妈看过来。
经常化妆卡粉的姐妹一定要试试。
正在减肥但坚持不下去的人，先别急。
```

### 2. Pain/Problem：引入场景痛点

目标人群圈住后，要把他们真实遇到的麻烦讲出来。

例如：

- 沙发底下灰尘扫不到。
- 洗碗手上总是油。
- 化妆卡粉、闭口、起皮。
- 减肥效果慢、坚持不住、越减越馋。
- 厨房油污擦半天擦不掉。

痛点要来自生活场景，不要一上来讲产品说明书。

### 3. Product：推出产品卖点

产品不是凭空出现，而是用来解决刚才的痛点。

卖点可以包括：

- 功能强。
- 效果快。
- 颜值高。
- 便宜。
- 数量多。
- 大牌同款。
- 使用方便。
- 口味好。
- 不伤手。
- 轻松省力。

卖点要讲清楚，并且要和痛点对应。

### 4. Promotion：促销行动

最后给用户一个现在下单的理由。

常见方式：

- 买一送一。
- 限时优惠。
- 拍两件更划算。
- 今天价格特别合适。
- 一条够用几个月。
- 折算下来一天只要几块钱。

这相当于直播里的逼单，但短视频里语气要自然，不要压迫感太强。

## 4P模板

```text
[目标人群]一定要看。
你是不是经常遇到[场景痛点]？

我最近发现了一个[产品]，
它可以帮你解决[具体问题]。

我最喜欢它的地方是[卖点1]，
而且[卖点2]，
用起来[真实感受/效果]。

现在[促销活动]，
算下来[价格感知]，
需要的可以去看一下。
```

## 顺序可以灵活调整

4P是基础结构，但不是死板顺序。课程提醒：拿到产品后，要先分析它最大的优势是什么。

如果产品功能很强，就把功能亮点放前面。

如果产品价格特别低，就把价格和促销放前面。

如果产品颜值很高，就先打颜值。

如果产品反差大，就先展示前后对比。

判断逻辑：

```text
产品最吸引人的点是什么？
用户最容易被什么点停下来？
那个点就可以前置。
```

## 带货文案常见错误

### 1. 信息密度太低

说了很多话，但只表达一个卖点。用户听半天不知道重点，容易划走。

改法：

- 一句话能说清楚，不要说三句。
- 每一句都要推动成交。
- 删除无效寒暄和废话。

### 2. 营销感太强

连续反问、命令式推销、过度夸张，会让用户产生被推销的不适感。

差：

```text
你还在为这个烦恼吗？你还不知道这个神器吗？
```

好：

```text
我最近找到一个不用手洗、还挺省力的小东西。
```

### 3. 卖点模糊

只说“很好用、很舒服、很划算”，但没有讲具体哪里好。

改法：

- 讲使用前后变化。
- 讲具体体验。
- 讲解决了什么麻烦。

### 4. 用词太专业

产品说明书式表达，用户听不懂。

差：

```text
采用全新生物酶配方，快速分解油脂。
```

好：

```text
这种厨房老油污，钢丝球擦半天都擦不掉，它喷两下，三秒就化开。
```

## 像朋友分享，而不是硬广推销

课程反复强调：好的带货视频不一定像推销，更像闺蜜、朋友、熟人把一个好东西告诉你。

要做到：

- 用真实使用感受说话。
- 先讲用户场景，不先讲产品参数。
- 把专业卖点翻译成生活语言。
- 不要一直坐着干说，要让画面动起来。
- 语气像分享，不像逼单。

## 让带货画面流动起来

抖音是视觉和听觉平台，不是只有文案。拍产品视频时，画面要有变化。

### 1. 边走边说

不要一直坐在一个地方干讲。可以一边走、一边拿产品、一边进入使用场景。

### 2. 直接进入使用场景

卖吸尘器，就把脏东西、纸屑、头发拍出来；卖清洁产品，就把厨房油污拍出来；卖帽子，就让用户看到戴上后的效果。

### 3. 多切细节镜头

包括：

- 产品特写。
- 使用前后对比。
- 手部动作。
- 细节质感。
- 使用过程。
- 结果展示。

### 4. 不要只坐着介绍参数

坐着干说会像广告，流动的画面更像真实分享。

## 欲扬先抑玩法

进阶玩法是“先抑后扬”：先提出怀疑、吐槽或反向观点，再通过真实测试转向产品优点。

它利用的是用户的好奇和八卦心理。

示例开头：

```text
这东西最近吹得太狠了，我买回来看看它到底值不值。
```

```text
这个东西用多了真的会出问题吗？我试了一下。
```

注意：

- 不要虚假贬损。
- 不要恶意攻击品牌。
- 最后必须回到真实体验和产品事实。

## 拿到产品后怎么做功课

### 1. 找5条以上同类带货视频

看别人怎么讲这个品，提炼可用角度。

### 2. 看商家详情页

详情页通常会写卖点、亮点、规格、优惠和使用场景。

### 3. 看用户评价

真实评价最有用户思维。好评里有用户为什么买，差评里有用户担心什么。

### 4. 看商品问答

问答区能看到用户真正疑惑的点，也能作为脚本痛点来源。

## 拍摄前准备

为了讲得自然：

- 真的使用产品。
- 熟悉产品优缺点。
- 重复练文案，练到能脱口而出。
- 开拍前调动状态。
- 找一个交流对象，像对朋友说。

不要僵硬地对着镜头念稿。

## 带货脚本检查表

```text
1. 有没有明确目标人群？
2. 有没有真实场景痛点？
3. 产品是不是为了解决痛点而出现？
4. 卖点有没有讲具体？
5. 有没有促销行动？
6. 信息密度是否够高？
7. 有没有强推销感？
8. 有没有专业词太多？
9. 画面有没有流动？
10. 语气像不像朋友分享？
```

## 行动清单

- 每个产品先写清目标人群。
- 每个产品整理至少5个真实使用痛点。
- 每个产品提炼3个核心卖点。
- 每条带货视频都用4P过一遍。
- 拍摄前看同类爆款、详情页、评价和问答。
- 产品视频必须有使用场景和细节镜头。
- 练到能自然分享，再开拍。
