# 视频学习 SOP

## 1. 采集信息

记录平台、账号、链接或文件名、发布时间、时长、品类或主题、已知数据表现。

## 2. 提取内容

- 口播文案：尽量转成文字。
- 屏幕文字：记录标题、字幕、贴纸、价格、卖点和评论区高赞反馈。
- 画面动作：记录开场画面、场景切换、展示方式、手势、表情、道具。
- 声音节奏：记录语速、停顿、情绪、音乐或音效。

## 3. 拆解结构

带货类优先拆：

- 前 3 秒钩子
- 痛点或欲望
- 产品卖点
- 证据和演示
- 价格或福利
- 行动指令

个人IP类优先拆：

- 人设信号
- 观点冲突
- 故事或经历
- 价值输出
- 情绪共鸣
- 关注理由

## 4. 提炼机制

每条视频至少回答：

- 它为什么能让人停下来？
- 它为什么能让人继续看？
- 它为什么能让人信？
- 它让用户下一步做什么？
- 我能借用哪个结构，而不是复制哪句话？

## 5. 入库沉淀

把核心字段填入对应的 `视频案例库.md`，并将完整拆解保存为单独案例笔记。
