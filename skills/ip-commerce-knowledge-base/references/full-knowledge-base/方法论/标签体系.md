# 标签体系

## 内容分类

- `#带货`
- `#个人IP`
- `#种草`
- `#测评`
- `#直播引流`
- `#知识输出`
- `#故事型`
- `#观点型`
- `#日常型`

## 钩子类型

- `反常识`：推翻用户原有认知。
- `痛点直击`：直接说出用户正在经历的问题。
- `结果承诺`：开头给出明确收益。
- `身份代入`：点名特定人群。
- `悬念缺口`：制造未完成信息。
- `对比冲突`：前后、贵贱、真假、好坏对比。
- `场景进入`：直接进入具体使用场景。

## 结构公式

- `痛点-方案-证据-行动`
- `场景-问题-产品-结果`
- `误区-真相-方法-提醒`
- `故事-转折-观点-关注理由`
- `身份-困境-经验-建议`
- `问题-清单-示范-总结`
