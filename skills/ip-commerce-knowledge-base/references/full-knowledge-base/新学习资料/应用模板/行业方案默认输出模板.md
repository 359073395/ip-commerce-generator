# 行业方案默认输出模板

## 使用场景

当用户提出：

- 我要做某行业IP。
- 我有某个产品想带货。
- 我想做实体店账号。
- 我想做个人IP加变现。

默认按本模板输出，而不是只给建议目录。

## 1. 信息判断

先判断信息是否足够。

如果不足，先补问最多 5 个关键问题：

1. 你卖什么，客单价多少？
2. 最想吸引哪类客户？
3. 你本人/品牌有什么背书和案例？
4. 能否真人出镜、直播、展示过程？
5. 用户看完后要去哪里成交：私信、到店、小黄车、直播还是私域？

如果用户不想填，先用合理假设生成“默认版方案”，并标注待确认项。

## 2. IP定位文档

输出格式：

```text
账号定位一句话：
我是[身份/经验/角色]，
专门帮助[目标人群]解决[核心痛点]，
用[内容方式/专业方法]让他们获得[结果]，
最终通过[产品/服务/咨询/课程/直播/带货]变现。

目标用户：
用户痛点：
用户潜在需求：
信任来源：
人设关系：
差异化：
内容边界：
合规边界：
```

## 3. 商业定位

```text
产品/服务：
客单价：
成交周期：
成交理由：
信任证明：
用户购买顾虑：
承接路径：
复购/升单路径：
```

## 4. 内容矩阵

默认四条线：

| 内容线 | 目的 | 适用脚本 | 示例方向 |
|---|---|---|---|
| 聊观点 | 吸真粉、立场、评论 | 观点型 | 替用户说话、批判行业乱象 |
| 教知识 | 显专业、收藏、精准粉 | 解题/案例/推荐/揭秘 | 高频误区、判断标准 |
| 晒过程 | 建信任、展示实力 | 过程/测评/挑战/体验 | 服务过程、制作过程、质检 |
| 讲故事 | 高信任转化 | 案例/小成就/苦难/平凡英雄 | 客户案例、个人经历 |

带货或实体店额外增加：

| 内容线 | 目的 | 示例方向 |
|---|---|---|
| 产品测评 | 眼见为实 | 横向/纵向/极限/化验 |
| 成交理由 | 降低购买顾虑 | 效果、性价比、服务、案例 |
| 直播预热 | 导入小黄车/直播间 | 福利、场景、问题解决 |

## 5. 选题生成

每个行业至少输出 30 个选题，按下面结构：

```text
普通选题
叠加爆款元素后的选题
适用脚本类型
内容目的
```

爆款元素优先使用：

- 成本。
- 人群。
- 奇葩/猎奇。
- 最差。
- 反差/对立。
- 怀旧/古代。
- 荷尔蒙。
- 头牌。

## 6. 脚本生成

默认给 3-5 条完整脚本。

格式：

```text
标题/角度：
目标人群：
脚本类型：
内容目的：
时长：

0-3秒开头：
正文口播：
屏幕字幕：
镜头/画面：
B-roll：
情绪刺点：
CTA：
风险提示：
```

## 7. 拍摄执行

输出：

- 第一帧画面。
- 场景。
- 道具。
- 服装/状态。
- 景别。
- 运镜。
- B-roll。
- 字幕重点。
- 封面标题。
- 剪辑节奏。

## 8. 运营节奏

默认输出：

- 7天起号计划。
- 30天内容节奏。
- 置顶视频建议。
- 直播/私域/小黄车承接。
- 评论区引导。

## 9. 复盘表

| 问题 | 判断指标 | 优化动作 |
|---|---|---|
| 播放低 | 2秒跳出高 | 第一帧、人群印记、钩子 |
| 完播低 | 5秒/整体完播低 | 信息密度、骨架、节奏 |
| 评论低 | 评论率低 | 观点、站队、提问、破绽 |
| 收藏低 | 收藏率低 | 步骤、清单、资料、工具 |
| 关注低 | 涨粉低 | 人设、长期价值、系列感 |
| 咨询低 | 私信/表单低 | 案例、信任、CTA |
| 成交低 | 点击/转化低 | 需求、证明、顾虑、促单 |

## 10. 最终交付顺序

1. 先给定位。
2. 再给内容矩阵。
3. 再给选题。
4. 再给脚本。
5. 再给拍摄清单。
6. 最后给运营复盘。
