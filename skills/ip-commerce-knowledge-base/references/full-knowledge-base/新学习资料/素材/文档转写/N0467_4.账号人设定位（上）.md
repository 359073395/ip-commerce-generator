# 4.账号人设定位（上）

账号定位=价值×用户×人设×类型×风格（账号定位>人设）

内核           形式

一：用户和价值

1.网红和IP的区别:

网红被消遣，IP被追随

网红服务粉丝，IP影响粉丝

网红无法为商品赋能，IP则可以为商品赋能

网红只有娱乐价值，IP的价值有无穷种具体的场景

（用户和价值的确定往往直接相关变现的方向，价值必须是实实在在的方法/资讯，而非只有价值观）

2.账号定位的两大误区：

（1）过度实用(赛道工具人)--用理念串联知识点、必须植入个人价值观，如“太史公曰:"

（2）过度抽象(价值不落地)-找到具体利他的内容分享生活和价值观(授课、采访、对话、事件、生活细节)

3.我给谁提供价值：（定义具体的人，一切都可以把握，一切都有判断标准）

（1）有明确变现目的的：

免费的精神

服务某些用户

（2）没明确变现目的的：（基于自身价值出发）（解决一个人一个阶段的一个问题）

服务某些用户

深耕某种价值

让账号聚集一批渴望某些事物的人（自己创造产品去卖/符合人群的产品）

A.怎么做选题：

长痘了怎么办

怎么和男朋友相处好好谈恋爱

怎么快速进入新行业学习

怎么适合自己职业                   照顾某个人

毕业了去哪个城市

怎么减肥

和妈妈观念不合怎么办

B.做视频考虑因素：

受教育水平

经济水平和消费水平

每天的生活状态和生活节奏

婚姻家庭状态、恋爱状态

他愿意付费的价值和购买力

有的时候认某一个死理的人也是一个人群

平时刷视频的习惯

此时此刻正在关心的问题

·反:否定与对抗，争取敌人的敌人

·合:交叉优势，立体竞争

·跨:跨界竞争、新的视野

·借:他山之石，可以攻玉

·弱:让自己的身体为自己说话增加分量
