# 26和27空间的处理和调度

空间是一个集合:物质、环境、光线、纵深、人群

空间的作用:

身份作用

情绪作用

品质作用

真实的情境(对消费域的刺激)

一：空间的基本布置（镜头里拍到的空间）

1.社会功能

利用场景的社会属性来交代身份

公开场合、私密场合

三维属性:视野、高矮、宽窄

视野中注意透气性

想开阔，镜头广，想窄，拍部分

内部材料:质地、肌理、数量、种类

质地想温馨的就拍柔软一点的东西（例如：抱枕、玩偶）

人想表达犀利：就要选择硬的东西

隔着玻璃拍人物

重复性元素

重复性元素极度强调主题 视觉饱满（多）

“弥漫”的介质

寻找动态的事物

动态事物容易让场景有氛围感

（例如：雪花、雨、烟雾、一群鸟、纱窗随风摆动、波光打在身上、光打在脸上．．．）

表意与象征

二：空间的层次与秩序

冷暖层次

服装、灯光．．．

明暗层次

二：空间的调度

多空间调度：安排情节并调动情绪
