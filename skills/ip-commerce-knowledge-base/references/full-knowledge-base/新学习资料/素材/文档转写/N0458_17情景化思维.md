# 17情景化思维

要在情境中去表达，包括文案也要对应情景和主题

情境化=人物+行动+对抗(关系)

对抗:人与人、大与内心人与环境(很多人)

１.最好的销售话术是包装使用场景，短视频天然的场景化表达的方式

２.从观众出发——从卖点出发

３.重构情境，所有需求都在情境中才能展开

４.文字：

事件（不要拍状态，要拍事件，状态模棱两可，感知不充分）

５．拍摄场景的关键是理解关系：

人与世界的关系

人与自己的关系

人与人的关系

表现力：

表现力是综合反应，不是单一表情因素，要看整体表现

先构建情景，在情境中表演
