# 15结构的精细化处理（骨架）

骨架：不同的骨架通过不同心理过程提高完播率并刺激互动(1:19)

转粉率和完播率非常重要

一：兴趣点推进:时刻感到新鲜有意思，来不及无聊，持续信息喂养

信息枚举型的优势（新鲜度）

·颗粒度很明显（节奏感）

·数字提醒有安全感

·显示结构，能引导观众往后看更多,即便有一些信息无趣也能让观众容忍

·起号推荐方式

即便不认识，也能高完播

利他

帮助我们了解节奏感

·本质也是一种选题的扩散

理念串联诸多点(多个卖点、多个知识点、多个产品)

让张小羊感兴趣的事件、知识点

让观众刷我们的视频就像刷抖音的不同视频一样

生活切片的聚合就是一个完整的视频（每个钩子可以组合成一个视频），非序列的

信息持续喂养，观众就来不及无聊

在情境中口播、在生活切片中口播

生活切片：

每一个事件的选择都是他生活中最精彩的部分（精简不啰嗦－讲高峰）

快速对每个切片中的事件进行总结

要有非常高的信息密度，每一个都是生活中的不平衡状态、事件

把最有表达欲的时刻讲出来（可以换场景、换知识点）

口播内容要融入环境，预埋情绪刺点（N+1）

卖点的聚合也是聚合（以有趣的方式呈现出来）（用画面讲产品，产品特点用画面展示）

所有视频的根本都是对信息密度的理解（要有内容的密度）

二：戏剧性推进:时刻感到紧张悬念并对结局有期待（过程比结果重要）

（1）四种可能：

·希区柯克式悬念:明知结局，但是想看怎么走到结局

·暗示了一个很值得期待的结局(很不可能、很惊艳等等)，但是行动屡屡受阻或者矛盾冲突升级或者扩散

·不断引出全新的信息，让情节重新被理解

·关心人物，深度移情(不容易使用)（例如：好人受难）

赌注1: 真的有悬念吗?你的观众真的关心吗?

赌注2: 关键场面是否引爆了?是否让观众的期待释放了

情绪刺点的预埋(履历、发心、性格特征)

三：共鸣型推进:时刻感觉到一种情感力量，时时刻刻看着对方想到自己（逐步共鸣）

共鸣推动的要素（一开始就要有趣）

·具象

我们通过具象的细节了解主角的生活，并且参与想象对号入座

·抽象

一句句抽象的句子指引着我们，提醒观众观看的方向

·结合

必要时候，三步一喘息

画龙点睛:用最终的升华和总结击碎心灵

具象为主(看热闹看故事)抽象为副(但画龙点睛)

（具体的是行动、事件、资讯等等；抽象的是观点、总结、感叹、想法、内心活动）

（共鸣是我们共鸣观众，而非要求观众共鸣我们）

一开始不要强行共鸣，先把新鲜事放在开头，再去讲共鸣的事件，才能独特

一开始要有趣，想办法让观众看下去

四：理性逻辑型推进:时刻有新认知，或者逻辑上反转，对此赞同或者不赞同

找一个人、找一个事、找一个东西来承载观点（先找不认同、再找认同）

（1）常见逻辑

·二元对立模式引发站队（先抑后扬）

观点直接对抗

例子证明，“无中生友”（我有个朋友、２０岁的我．．．）

·不断颠覆，不断轰炸认知

不需要教学，而是单方面告知

·从现象到本质，从特殊情况到一般情况

·三段论灵活组合(是什么-为什么-怎么办)

为什么-怎么办

为什么-是什么-怎么办

是什么-怎么办

五：节奏型推进:时刻感到律动且有“洗脑”感，跟着旋律和鼓点看完

卡点（音乐）  排比  押韵  长短句  内在的气

注意力管理意识

·我必须知道我时刻让观众看下去的理由是什么

从“单方面的输出“到”时刻担心观众什么时候会看不下去”

·你知道观众什么时候会无聊，什么时候压力过大

适时总结

适时聊天

适时打趣

适时变调

适时缓解疲劳

注意疏密有致，张弛有度

·时刻有体谅之心，别只顾着自己讲

颗粒度意识（转折就是最好的颗粒度，但是、因为....）

·主题是线，信息之间要有颗粒度，颗粒之间不要粘连

·不要想着过度自然，学会大胆跳跃

·不同的事件层次感非常清楚

·不同的知识点和信息点，也能让别人感觉信息密度很强

·有必要解释的明确的递进关系，而不是靠感悟的微妙关系

势能意识

任何时间的艺术都是积累和释放的艺术，素材的排列避免平行排布

要构建出素材的势能，层层递进推向终点

任何不强相关的信息都会削弱势能

高情绪状态人们更容易点赞评论

刺激观众 冲击情绪阈值

·大多数人在刷抖音时是相对麻木的甚至是疲惫的，不可能提前酝酿情绪好情绪

来准备来看的视频

·点燃观众到一百度的沸腾的热水，只有刺激观众让观众到达某个情绪阈值，才

能触发观众的反应

·不同行为阈值不一样

点赞最低、收藏次之

评论和分享比较高

关注阈值相对来说比较高
