# 11.表现力

表现力的迷思

●“我为什么会说话”

●“我在什么动作状态下说话”

·在行动中

让人物在行动流中行动，创作者捕捉生活流中的一个段落

·动作的停顿

动作的停顿意味着突发或关键点。停顿，可以抓取观众的注意力，形成节奏的变化。

·动作的叠加

通过人物多线程操作来增强真实性

·动作的方向

每一种方向和转向，都有意义

·本身无意义的动作

通过人物的动作带出环境、关键道具等等。
