# 5.账号人设定位（中）

一：人设（接人气、谈论自己是谁、情商和做人的问题、学会自卖自夸）

人设是观众对我们印象的综合

人设是一种营销，而非介绍

一切品牌商家个人化，一切职业身份个人化

4.个人化、势在必行

因为赛道拥挤同质化(标签系统可破)

因为商业气息过浓(广告惹人生厌 软提及容易接受)

因为爆炸性增长红利消失(人设强度深度信任)

因为铁粉机制限制(保持新鲜可持续发展)

4.如何挖掘自己身上的营销点：

（1）寻找身上的故事性

你的形象特征和你有什么格格不入的地方?

你的过去的经历中有什么戏剧性的地方?

你的履历里是否有什么拿得出手的部分?

你的爱好和你的职业有没有什么分裂之处?

（2）质问自己的性格、品行、人格魅力

共情力强，善于打动人

虽然逻辑能力强，但讲话有温度

行业资深，但是说话易懂

身居高位然而特别没架子

特别仗义

特别疼老婆

（3）摆正自己的姿态确认和观众的关系

提醒者

服务者

娱乐者

领导者

陌生人

线上恋人

私密知音

联系客服

萍水相逢的陌生人

知识分享型：（放入具体场景中）

1.开宗:提出新的领域、构建新的思维方式、创造全新的需求、创造全新的品类

2.立派:在特定领域中构建新的概念和流派、在品类中创造新的需求侧重点

3.高徒与传人:坚守某种风格并不断演绎与传播、在需求链条上做垂直细分

4.同道中人、灵魂伴侣、师哥师姐:略高一筹的同行者、同质化中强调性比价

5.商贩、手艺人、艺人、书童:单纯的向上提供者，不知名的产品矩阵

（4）梳理自己的三观你所认可的和你坚决反对的（写在简介里）

世界观
