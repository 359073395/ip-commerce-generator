# 7.常规人设内容以及作业模板

公共话题+私人体验（让成为故事的亲历者：这个故事因为你产生影响）

普世+陌生

新鲜资讯与洞察（基于用户来看）信息差是绝对的流量密码

行业内幕揭露

行业最新动态

“我”看”你”的行业

二.认知真相（我为你说出你想听的话，不敢说!说不出!刷不到!真相!）

·关键在于掏心掏肺

·关键在于戳破大家的面具

·关键在于提供真相

·关键在于敢说

·关键在于颠覆

·关键在于内幕揭露

（强化对方的偏见、想法、信念，调动对方的情绪，观念......）

三. 情感深处

·自我检讨/批判/忏悔

·对身边人的极致坦诚与私密谈话

（在坦白中展现美好和人设）

四.深爱世界

·对身边人(事物)的夸奖与感恩

·为他人做一些真正对他有意义的事情

·我如此深爱这个世界

·特殊日子的记录

五.场景中的刚需解决

·关键!具体场景!越具体越好!（所有的痛点、需求要在具体场景实现）

·关键!尽量高频!

·关键!真实痛点!

·关键!真正得到解决!

·进阶!解决方法的新鲜度!(必要时刻为之命名)

六.精彩的过程（用创作者角度看、发现问题、寻找思路）

·期待的构建

·不可能完成的任务

·值得期待的结果

·令人向往的事件

·精彩的来由

·过程的本身的可看性

·障碍不断出现

·时间的压缩

·信息量的补充(*的注解)

·轻知识

·内心OS

·旁白的解读
