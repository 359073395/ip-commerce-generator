# 薛辉视频短视频剪辑

_注：该 DOCX 的媒体文件可能损坏，已使用 XML 正文兜底提取。_

薛辉视频短视频剪辑

工具栏:

动画:入场动画:素材开始时的动画效果

出场动画:素材结束时的动画效果

音频分离:将素材的声音单独建立一个轨道，把原本的画面删除，换成新的画面，声音还是用原来的。

编辑:镜像:将选中素材画面左右翻转。

旋转:将选中素材90度旋转

裁剪:改变选中素材画面大小。

滤镜:对比度、亮度，尽量不要超过25%。

蒙版:就类似视频是一张纸，我在他前面在盖上一张纸，如果我想看视频是圆的，我就在前边盖的纸上抠个圆形，这样看到后边的视频就是圆形的了。

不透明度:修改所选中素材的不透明度，数值越大画面越实，数值越小画面越透明。

倒放:将选中素材倒放，比如讲3天前，一个小时前。

定格:将选中素材中制定的画面进行静止显示。把选中的视频变成图片定格3秒钟，就相当于一个截图。

画中画:将图片或者视频素材，以在画面上，增加画面的方式呈现出来。比如减肥对比图，比如装修前后图。

二:剪辑

1.剪掉气口:节奏更紧凑，把字幕识别后，把中间没有字幕的地方默认气口，分割后剪掉。

文本:一行不要超过8个字，不要用2行，把一段长的字幕分割成几段。

操作:我一学生，把“生”的部分点击切割，把生后面的其他字全部删掉，“一天视频”后字   幕开始继续分割，然后删除前后多余的字。

常用字体

如果想文本中某几个字变颜色比较醒目操作:比如“抖音搜索”我想让搜索两个字变颜色，变大，找到文字双击——在对话框长选择“搜索”两个字——点击“编辑样式”就可以操作了。

总结:

3:关键帧:距离越远，变化越慢，距离越近，变化越快，显得情绪更激烈。

放大或缩小画面:开头先放一个关键帧，然后拉到视频后边，点击关键帧，手动放大或缩小  画面，关键帧就形成了。

调解饱和度:

调节贴纸的位置:从左到右

调节字幕:从下到上

画中画:在选好的位置点击画中画——导入提前选好的视频——调节画中画的长度大小——然后点击贴纸搜索手机框——调整贴纸的长度与画中画相匹配。

蒙版:选择蒙版，将蒙版移到想要蒙住的位置，左上角的图标往外拉，可以把正方形的角变圆，下边的图标可以虚化边框，剩下的两个调节大小高矮。
