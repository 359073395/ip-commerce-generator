# 21故事性思维

人群印记：不仅仅是指某个赛道，也值人物出场的性格、说话语气、背景音乐等

情境是故事中的一个切片

切片聚合（并列的切片）

（１）共同的主体，把切片聚合在一起，没有因果关系，只是单纯的聚合在一起

（２）切片要有共同对抗的东西—共同的敌人（同样的价值对抗，构成同样的主题）

（３）统一的正负价值对抗，但凡和对抗无关的、和主题无关的都不要

一定要明白对抗的点是什么、对抗的价值观是什么

人与人的对抗、人与内心的对抗、人与世界的对抗

有强主角成长故事（一环扣一环：因为Ａ产生Ｂ、因为Ｂ产生Ａ　）

人物登场—特定环境中人物

激励事件—激发欲望，引发行动

故事脊柱—出现障碍、形成对抗

故事结局—对抗结束（邪不压正、道高一尺，魔高一丈）

故事的内核（对抗）

（１）人物

人物=欲望=价值载体

人物是欲望的载体

人物是价值的载体

（２）激励事件

●生活一般都在动态平衡中

●有事儿发生，要么被迫改变，要么主动改变，主角必须产生行动

“每天都做饭，今天还做饭”，这不叫激励事件

“平时都是吃外卖让身体很糟糕，但是医生告诉你不改变不行了，逼着你改变”，这叫激励事件

●激励事件中我们看到了人物的渴望和观众的共同困境

问题和危机的出现是故事的开始

大多数时候为了高效叙事，直接就是问题和危机出现而非舒缓的人物登场

故事脊柱（正负价值的对抗）

Ａ．实验过程

Ｂ．思探索过程

Ｃ．创作过程

Ｄ．服务过程

故事的过程在于不断的构建对抗，不断的塑造困难（构建行动线的关键是寻找障碍）

故事成不成的关键在于对抗

重要的是故事走的进程本身，不是人物、不是结局、不是价值观

在正负价值激烈对抗中产生情感的激荡，从而检验主角(人物/产品/观点)的品质

故事的技法

构建强大的障碍是故事有力的关键

过程精彩会冲击情绪阈值，高情绪状态下人转化成了感性判断

故事的发展发向

1.社会进展:横向蔓延到更多人甚至整个社会(小杨哥)

2.个人进展:深入人物的私人生活和内心世界中(自媒体常用)(海口)

3.象征升华:从个体到诸多个体的普遍现象(躺平)

4.反讽升华:这些故事的主人公都自认为他们确切地知道他们必须做什么，而且具有一个精确的行动计划。他们认为生活就是

A、B、C、D、E。正是在这种时候，生活却喜欢把你弄得晕头转向，踢着你的屁股，坏笑道:“今天不行，我的朋友。今天是

E、D、C、B、A。对不起。”

1.他终于得到了他一直想要的东西…但是已经太晚，他不可能拥有它。

2.他被推到离他的目标越来越远的地方……结果却发现事实上他已被引导到他的目标。

3.他抛弃了他事后才发现对他的幸福不可缺少的东西。

4.为了达到某一目标，他不知不觉地采取了一些背道而驰的步骤。

5.他采取行动想要毁灭某一事物，结果却适得其反，搬起石头砸了自己的脚。

6.他得到了某种他坚信会给他带来厄运的东西，想方设法要摆脱它，结果却发现那是一份幸福的

（３）故事的关键场面

1. 灵魂黑夜的倾诉

2.纵身一跃的勇气

3. 决战时刻的坚定和胜利
