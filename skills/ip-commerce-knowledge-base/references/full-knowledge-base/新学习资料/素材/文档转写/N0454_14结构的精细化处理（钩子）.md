# 14结构的精细化处理（钩子）

钩子-骨架-情绪刺点

钩子不一定只出现在开头，骨架不一定只出现钩子之后

钩子:“强节奏” 骨架:节奏型推进

钩子:反常(不可能完成的任务)，关注刺点:能人所不能的人设

理解为“动力结构”而非”模版结构”

钩子可以叠加、骨架可以叠加、情绪刺点可以到处预埋

一：13种钩子

（可以语言表达 也可以视觉表达互相融合交叉叠加）

1.不平衡情绪登场(最重要最有效)-切片性

●表情高情绪

·个人带着故事登场(肉麻、咆哮、痛苦、笑容、魔术师口吻:煞有其事，小题大作，我有特别牛的事情要讲)

●故事高情绪

·故事中的人与人之间的强冲突

·突发状况

●镜头画面高情绪

·明艳的色彩、快速变化的镜头(焦段变化、运动变化、剪辑变化)

●动作高情绪

·极其肉麻的动作

·极其愤怒的动作

●哪怕只是简单的一句惊叹

·真刺激啊!

·这也太好吃了!

·不是吧!这么值!

（情绪冲击和卷入）

2.完形心理

·并置陌生事物(语言或视觉)

《枪炮病菌与钢铁》/《菊与刀》

·构建关系并让观众吃瓜(有的时候是对抗、有的时候是结合)

3.与我有关（和观众建立关系）（回到顾客的使用场景）

·点名人群（直接属性、女孩们、金牛座们.......）

·直接对话/互动

你看.

给你三秒钟..

·主题诱人(故事中或标题中，类似人群印记)

·生活中熟悉的疼痛和美好的可能(兴趣电商)（视频给人一种就是与我有关）

痛点式包装（不喝咖啡不能工作）

直接展示正向结果（就喜欢这种宁静的美好、直接衣服遮住大肚子）

4.否定性力量（损失厌恶以及对未知的恐惧）

·重大盲区

·危害纠错

·常识颠覆

撕开认知缺口

人们总是认为自己认知完整

5.社会身份的敏感

●挑衅进而引起愤怒和骄傲

·站队

·拉帮结派

·强行捧踩

·自我吹嘘

·攻击某类群体

●自我贬低以换取观众的优越感(当然有的人并不认为自己在自我贬低)

·犯傻、翻车、沙雕、土味

·傻的不自知、淳朴、单纯、简单

·暴露自我弱点

·特别困难但是特别热爱生活

·特别朴素但是却出口不凡

●这是双刃剑

●食欲

●情欲

●听觉

·沉浸式

●视觉

·跳跃的信息

·强迫症倾向

·闪闪发光的事物

·触觉唤醒

●触觉(物的表达)

·极度舒适、解压或者引发鸡皮疙瘩

●危险和疼痛预警

·烫伤、刺疼等等

·人多

·不可能完成的任务(小学学历就可以去清华北大演讲等等)
