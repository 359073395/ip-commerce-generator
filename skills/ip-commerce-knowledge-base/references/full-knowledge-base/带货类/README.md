# 带货类知识库

带货类内容的核心目标是让用户从“注意到”走到“愿意买/愿意点/愿意问”。

## 重点观察

- 商品解决了什么具体问题。
- 开头是否快速进入痛点或欲望。
- 卖点是否被演示出来，而不是只说出来。
- 证据是否可信：实拍、对比、用户反馈、数据、场景。
- CTA 是否明确：点小黄车、进直播间、评论关键词、领取福利。

## 常见内容类型

- 产品测评
- 使用演示
- 前后对比
- 清单推荐
- 场景种草
- 直播间切片
- 用户痛点解法
- 价格福利型短视频
