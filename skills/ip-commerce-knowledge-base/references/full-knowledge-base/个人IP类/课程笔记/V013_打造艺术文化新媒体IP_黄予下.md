## 一句话结论

个人IP最终不是为了当网红，而是用新媒体放大信任，再用产品、服务和私域承接流量，成为能赋能行业的商业IP。

## 核心框架

### 1. 放下面子，先下场

黄予用联通、建设银行、消防等机构做短视频的例子说明：连国企和严肃机构都在下场做内容，专业人士更不必把“面子”当成阻碍。

可复用洞察：

- 做IP的第一关是心理建设。
- 富人要结果，穷人才只要面子。
- 专业身份不是不做短视频的理由，而是短视频里的信任资产。

### 2. 三条置顶是信任名片

他再次强调起号先做三条置顶：

- 我的20年：我是谁，我长期做什么。
- 我20年里最引以为傲的事。
- 我近几年做得最漂亮的一件事。

可复用洞察：

- 置顶不是随便放爆款，而是解决陌生用户信任。
- 黑粉进入主页后，如果看到完整资历和结果，可能转为中立甚至信任。
- 高客单IP尤其需要把信任证据前置。

### 3. 蹭热点公式

课程给出热点内容公式：

```text
下钩子 -> 蹭热点 -> 引入自己的产品/故事 -> 总结热点 -> 引导自己的产品 -> 评论区互动
```

黄予用董宇辉、胖东来等热点，最后都落到自己的“服务”和“艺术收藏服务”上。

可复用洞察：

- 蹭热点不是复述新闻，而是借热点讲自己的价值观和产品逻辑。
- 热点要和自身业务有连接点。
- 结尾要自然引导用户讨论或咨询。

### 4. 内容公式：下钩子、拉停留、输专业

另一个通用公式：

```text
下钩子 -> 拉停留 -> 输出专业知识或产品设计 -> 埋槽点/问题 -> 评论区互动
```

案例里，原视频开头太平，播放约19万；调整后用“一面墙就值好几千万”作为钩子，播放破百万。

可复用洞察：

- 开头黄金几秒决定用户是否继续看。
- 先给强钩子，再把精彩片段前置拉停留。
- 专业内容也要先让用户愿意留下来。

### 5. 泛流量可以盘活账号

课程提到陪跑团队建议：一周做 5 条内容，3 条专业，2 条泛流量。泛流量可以盘活账号，提高整体播放和推荐。

可复用洞察：

- 不要抗拒泛流量，关键是泛流量仍然服务人设。
- 泛流量负责盘活，专业内容负责沉淀和转化。
- 新媒体平台本质有娱乐属性，太专业会没人看。

### 6. 正确“抄结构”

黄予用“教路人唱歌”改成“教路人学当代艺术”的案例说明：可以借用爆款结构，但要换成自己的专业和场景。

可复用洞察：

- 正确的抄是抄结构、节奏和互动设计。
- 错误的抄会破坏人设。
- 任何爆款结构都要先判断是否符合自己的人设定位。

### 7. IP人格定位

课程引用 12 种人格原型，并转化成做IP的四类：

- 教导者。
- 陪伴者。
- 崇拜者。
- 搞笑者。

可复用洞察：

- IP方向定错，后面内容越做越别扭。
- 不能乱组合人设，否则用户记不住。
- 做IP要追求鲜明，而不是完美。

### 8. 流量是放大器，产品才赚钱

课程后半段强调：流量是放大器，产品是接住流量的容器。优秀IP + 极致社群 + 顶级产品，等于一个超级公司。

可复用洞察：

- 不要只追求成为网红，要成为商业IP。
- 公域流量最终要进入私域和社群。
- 没有产品，流量再多也只是热闹。

### 9. 新媒体 + 个人IP + AI

黄予总结：18年前所有行业值得用淘宝做一遍，11年前值得用微商做一遍，8年前值得用新媒体做一遍，现在所有行业值得用AI做一遍。

可复用洞察：

- AI是当前个人IP的效率工具。
- AI不能代替艺术、审美、专业判断和真实服务。
- 知道不等于做到，行动才是真知。

## 可复用结构

```text
放下面子 -> 三条置顶 -> 蹭热点 -> 下钩子拉停留 -> 泛流量盘活 -> 抄结构 -> 定人设 -> 产品承接 -> AI提效
```

## 适合改编的选题

1. 专业人士做IP，第一步不是拍视频，而是放下面子。
2. 三条置顶视频，决定陌生人信不信你。
3. 蹭热点最高级的方式，是讲回自己的产品和价值观。
4. 泛流量不是没价值，它能盘活你的账号。
5. 流量是放大器，产品才是赚钱的“器”。

## 风险与注意

- 艺术品和投资相关内容不能做收益承诺。
- “抄结构”需要原创改写，不能搬运。
- AI工具只能提效，不能替代真实专业和交付。
