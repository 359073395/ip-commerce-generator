## 一句话结论

素人 IP 起盘不要一开始就死磕垂直变现内容，先用更容易爆的泛内容把账号做活，再按阶段调整泛流量、垂直内容、直播引流和私域承接的比例。

## 课程定位

这一节是玺颜“素人 IP 陪跑方法论”的第一部分，重点讲新盘起号、内容比例、对标调研、直播引流和项目节奏。后续 V003/V004 属于同一系列的延伸和问答。

## 核心框架

```text
前期准备 -> 内容展现形式 -> 直播承接 -> 项目节奏把控 -> 私域/变现团队跟进
```

## 关键拆解

### 1. 做流量没那么神秘，先进入正确路径

玺颜强调，真正厉害的操盘手不是只把自己的账号做起来，而是能带团队、学员和客户也拿到结果。

可复用洞察：

- 很多流量问题不是玄学，而是路径和节奏错了。
- 文件、模板和细节只是工具，关键是有人能判断你是不是走在正确路上。

### 2. 前期准备：团队必须知道项目好在哪里

起盘前，IP 本人、操盘手和流量团队都要清楚：

- 行业好在哪里。
- 项目好在哪里。
- 这件事能给别人带来什么价值。
- 目标用户是谁。
- IP 本人的经历、结果、性格和可表达素材有哪些。

可复用动作：

- 做行业调研。
- 做 IP 个人资料分析。
- 做项目整体规划。
- 让 IP 有仪式感和信念感。

### 3. 起号早期不要只追求垂直流量

课程中很重要的一点是：前期不是只发垂直口播内容。更快让账号活起来的方式，是泛内容和垂直内容按比例组合。

参考节奏：

- 1.0 阶段：泛流量内容约 70%，垂直内容约 30%，先让账号活起来。
- 2.0 阶段：泛流量和垂直内容约 50/50。
- 变现强化阶段：垂直变现内容约 70%，泛流量内容约 30%。

可复用洞察：

- 账号没活起来前，过早追求精准变现，可能导致流量起不来。
- 泛内容不是没用，它负责拉账号活跃、带来进入主页和直播间的人。

### 4. 对标不能只找同行

很多人说找不到对标，是因为只找同行。玺颜建议拆得更细：

- 内容对标。
- 行业对标。
- 人群对标。
- 拍摄形式对标。

如果同行里找不到，可以先明确目标客户的年龄、性别、喜好、痛点，再去找其他行业里粉丝人群相似的账号，借鉴它们的拍摄形式和内容结构。

可复用洞察：

- 同人群跨行业对标，比死盯同行更容易找到可用素材。
- 对标不是复制，而是拆“人群为什么停留、为什么互动、为什么进主页”。

### 5. 三类高性价比内容

课程提到几类适合起号和主页承接的视频：

- 秀结果：展示自己或客户拿到的结果。
- 人设故事：讲经历、身份、家庭、转折和可信度。
- 互动/情绪类：用一句话、强情绪、反差、争议或互动激发转发和评论。

置顶视频尤其重要，因为爆款视频把人带进主页后，用户最先看到的是简介和置顶内容。

可复用洞察：

- 置顶视频要承担“我是谁、我凭什么、我能带你去哪”的作用。
- 变现阶段要把能建立信任和证明结果的视频放在最显眼的位置。

### 6. 短视频引私域变难，直播承接更重要

课程提到，现在用户被太多人私信教育过，单靠短视频私信引流的效率下降。更有效的方式是：

- 先用相对宽泛的视频获得流量。
- 视频爆了以后马上开直播承接。
- 在直播间建立粘性，再引导进入私域。
- 用直播素材二次剪辑或回放继续承接。

可复用洞察：

- 短视频负责拉人，直播负责建立关系和筛选意向。
- 对一些案例来说，直播/回放素材能承担主要引流，但要注意平台规则和素材风险。

### 7. 私域团队不要过早搭

玺颜提醒：没流量之前，不必先重金搭私域团队。前期老板或小团队自己能接住，就先接住。

更合理的顺序：

```text
先找到爆款流量 -> 再优化引流动作 -> 再搭私域/销售团队 -> 再稳定放大
```

可复用洞察：

- 有客户进场后再搭销售团队，避免组织成本前置。
- 没有稳定流量时，私域团队容易闲置或动作变形。

### 8. 四个项目阶段

课程中把项目推进分成几个阶段：

- 1.0：找能爆的内容形式，让账号活起来。
- 2.0：持续优化引流动作。
- 3.0：按内容布局和节奏发布，开启直播承接。
- 4.0：进入稳定期后，做多元化整盘设计。

可复用洞察：

- 每个阶段工作重点不同，不能一开始就把所有动作同时上。
- 先爆内容，再开直播，再重私域，再做矩阵或整盘升级。

## 适合改编的选题

1. 素人 IP 起号，为什么前期不能只发垂直口播。
2. 找不到同行对标时，去找同人群跨行业对标。
3. 账号没活之前，别急着花钱搭私域团队。
4. 爆款视频之后，主页置顶到底该放什么。
5. 短视频引流变难后，直播间才是新的私域入口。

## 风险与注意

- 课程中播放量、直播间人数、转化比例等数据为讲师口述，未外部验证。
- “抄对标”应理解为拆结构和策略，不能复制具体文案、画面和表达。
- 涉及录播、敏感赛道、公域引流等动作时，应优先遵守平台规则。
