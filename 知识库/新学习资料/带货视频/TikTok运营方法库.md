## 核心定位

TikTok电商不是传统“人找货”的搜索电商，而是“货找人”的兴趣电商。

短视频内容的作用是：

```text
内容钩子 -> 激发兴趣 -> 建立信任 -> 引导点击商品卡/小黄车 -> 商品卡承接 -> 成交
```

所以 TikTok 带货要同时优化三件事：

- 内容数据：完播、互动、分享、点击。
- 商品数据：CTR、CVR、评价、价格、详情页。
- 商业数据：GMV、GPM、CPA、ROAS、ROI。

## 流量逻辑

TikTok采用流量池赛马机制。

常见流量池：

- 基础流量池：200-500。
- 一般流量池：1000-5000。
- 中级流量池：10000-50000。
- 高级流量池：100000-500000。
- 终极流量池：100万以上。

核心权重：

```text
完播率 > 互动率 > 点击率
```

带货视频进入精准商业流量池后：

```text
GPM / 转化成本 / 商品卡承接能力
```

会成为后续推荐的重要指标。

## 核心数据指标

### 短视频表现

| 指标 | 含义 | 优化方向 |
|---|---|---|
| 播放量 | 初始曝光 | 不单独迷信播放 |
| 完播率 | 内容是否留人 | 优先优化前3秒和整体结构 |
| 点赞率 | 初步认可 | 情绪、爽点、共鸣 |
| 评论率 | 话题性 | 提问、站队、争议 |
| 转发率 | 社交货币 | 有用、有趣、能分享 |
| 吸粉率 | 长期价值 | 人设、系列、持续期待 |

### 商品与店铺

| 指标 | 含义 | 优化方向 |
|---|---|---|
| CTR | 商品卡点击率 | 主图、标题、价格、视频引导 |
| CVR | 商品卡转化率 | 详情页、评价、价格、促销、信任 |
| AOV | 客单价 | 套装、加购、定价 |
| 退款率 | 售后健康度 | 产品质量、承诺准确 |

### 广告投放

| 指标 | 含义 |
|---|---|
| CPM | 千次展示成本 |
| CPC | 点击成本 |
| CPA | 成交成本 |
| ROAS | 广告销售额 / 广告花费 |

初期重点：

- 内容阶段看完播和互动。
- 带货阶段看 CTR。
- 商品承接阶段看 CVR。
- 放量阶段看 GPM、CPA、ROAS。

## 爆款短视频结构

### 1. 黄金3秒

公式：

```text
精准人群 + 剧烈痛点 + 视觉冲击
```

可用画面：

- 真人痛苦表情特写。
- 问题场景再现。
- 产品效果瞬间。
- 对比冲击画面。

可用字幕：

- “30岁以上必看？”
- “如果你还在这样用/吃/买，就错了。”
- “[目标人群]注意，这个坑很多人都踩过。”

避免：

- 慢悠悠风景。
- 无关动画。
- 品牌Logo开头。
- 没有用户场景的空镜。

### 2. 痛点共鸣

5-10秒内深化痛点，让用户觉得“说的就是我”。

### 3. 产品引入与信任

证据可选：

- 成分/材料/功能特写。
- 权威背书。
- 专利、检测报告、研究材料。
- 用户好评截图。
- 使用前后对比。
- 真实测评过程。

### 4. 转化引导

不要只说“半价”，要把价格重塑为用户熟悉的成本：

```text
一顿外卖的钱
一杯咖啡的钱
每天几毛钱
一次试错成本
```

同时加入：

- 限时。
- 限量。
- 前N名优惠。
- 退货/售后保障。
- 赠品。
- 下单指令。

## 商品卡优化

商品卡是卖货最后一站，决定用户点进来后是否成交。

平台会综合评估：

- 商品信息完整度。
- 标题关键词。
- 主图质量。
- 价格竞争力。
- 评价数量和质量。
- 点击率 CTR。
- 转化率 CVR。
- GMV/GPM。

### 诊断逻辑

```text
高曝光 + 低CTR = 主图/标题/价格不吸引
高点击 + 低CVR = 详情页/评论/价格/信任不够
视频爆了但不成交 = 商品卡承接弱
```

### 商品卡图片结构

- 第一张：强主图，清楚展示产品和核心卖点。
- 第二张：卖点对比图。
- 第三张：细节特写。
- 第四张：功能或组件展示。
- 第五张：尺寸参照。
- 第六/七张：使用场景。
- 第八张：15秒短视频。
- 第九/十张：包装、认证、品牌、信任证明。

## GMV Max投放

GMV Max更适合已有基础数据的商品，不适合0销量、0评价、无学习数据的新商品直接放量。

启动前检查：

- 商品已自然出单或有基础转化。
- 商品卡已优化。
- 有评价和基础信任。
- 有可用视频素材。
- ROI目标合理。

### 判断投放问题

```text
花不出去 + ROI低 = 素材弱/商品弱/系统无学习数据
预算能花完 + 转化差 = 商品卡、价格、详情页或视频引导弱
CTR低 = 素材点击吸引力弱
CVR低 = 商品卡承接弱
```

GMV Max不要频繁大改，重大调整会让系统重新学习。

## 达人邀约与运营

达人合作不要只看粉丝量，要看：

- 近30日GMV。
- 平均播放量。
- 粉丝画像。
- 评论区质量。
- 是否卖过类似品。
- 历史带货稳定性。

### 达人合作逻辑

- 新店破零：靠达人内容快速带动订单和店铺权重。
- 信任背书：本土达人比商家自夸更可信。
- 内容外包：达人提供本土化表达和素材。
- 算法加持：达人优质内容也可能被系统二次推荐。

### 合作原则

- 广撒网，批量测试。
- 纯佣或低成本寄样起步。
- 80%的预算和时间放到20%已验证达人。
- 多夸、多反馈、多报喜，让达人有成就感。
- 不要把希望压在单个达人身上。

## TikTok短视频SOP

九环节：

1. 明确目标：产品、受众、卖点、考核指标。
2. 寻找对标：竞品、达人、同品类爆款。
3. 拆解爆款：钩子、结构、BGM、信任、CTA。
4. 整理素材库：痛点场景、产品特写、评价、证据。
5. 视频剪辑：标准化流水线生产。
6. 视频发布：标题、标签、商品卡关联、价格核对。
7. 数据观测：完播、互动、CTR、CVR、GMV。
8. 复盘优化：定位问题并决定下一轮动作。
9. 爆款复制：横向、纵向、矩阵复制。

## 爆款复制

### 横向复制

同一产品或同功效产品，换：

- 演员。
- 场景。
- BGM。
- 开头。
- 口播版本。

### 纵向复制

把已验证的视频结构复用到同品类其他产品。

### 矩阵复制

将爆款视频投流加热，并作为信息流广告素材扩大收益。

## AI工作流

AI不是替代者，而是副驾。

AI驱动流程：

```text
输入指令 -> 生成多个脚本/标题/创意 -> 人工筛选优化 -> 拍摄剪辑 -> 发布 -> 数据复盘 -> 反馈AI优化下一批指令
```

### 指令公式

```text
任务 + 背景信息 + 格式要求
```

基础脚本指令：

```text
请针对一款【产品名称】创作一个TikTok短视频带货脚本。
主要卖点包括【卖点1】【卖点2】【卖点3】。
目标受众是【国家/地区 + 人群】。
脚本需要包含强开头钩子、痛点展示、产品解决方案、信任背书、限时行动号召。
请用【语言】输出，并要求口语化表达。
```

迭代指令：

```text
这是上一个脚本的数据：点击率高但转化率低。
请分析可能原因，并生成一个优化版本，重点加强信任背书。
```

AI可用于：

- 脚本多版本。
- 标题。
- 长尾关键词。
- 分镜创意。
- 封面文案。
- AI配音。
- 数据复盘。
- 提示词库沉淀。

注意：

- 产品参数必须人工事实审核。
- 功效和承诺必须合规审核。
- 本土化表达必须人工把关。

## 复盘优先级

当不知道先优化什么，按顺序排查：

1. 开头3秒：解决完播问题。
2. 互动点：解决点赞、评论、分享。
3. 引导话术和购物车卡片：解决CTR。
4. 产品详情页和价格：解决CVR。
5. 广告和达人：解决放量。

## TikTok运营成长阶梯

### 新手期 0-6个月

目标：做合格执行者。

- 熟悉CapCut、Seller Center。
- 1:1模仿对标账号。
- 会记录基础数据。
- 建立闭环：执行、反馈、复盘。

### 成长期 6-18个月

目标：成为优化者。

- 能分析完播、互动、CTR、CVR。
- 能做A/B测试。
- 能做基础投流。
- 能初步做达人合作。

### 成熟期 18-36个月

目标：建立增长体系。

- 建SOP。
- 管理团队。
- 统筹短视频、直播、投流、达人、货盘。
- 关注利润率、CAC、LTV。

### 战略期 3年+

目标：成为行业专家。

- 预判平台政策和市场变化。
- 整合供应链、资本、流量。
- 建品牌。
- 输出方法论。
