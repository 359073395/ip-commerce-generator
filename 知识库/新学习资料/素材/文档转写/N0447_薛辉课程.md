# 薛辉课程

_注：该 DOCX 的媒体文件可能损坏，已使用 XML 正文兜底提取。_

薛辉课程

一:

当你知道你卖什么了，你就知道你要拍什么了。

搭建素材库

一:素材库的五个文件夹:选题，脚本结构，开篇结尾，呈现方式，音乐。这个五个大箱子，然后每个大箱子里面再弄个具体的小分类。

开篇结尾（金句结尾）:潇洒、金钱、警告、励志、爱情、亲情、友情。

二:搜索:

选题搜索:抖音、快手、视频号。

内容搜索:知乎、小红书。

三:

选题:关键词、巨量算数、热点。

百度:行业内词+关键词（时间、地点、人群、行业细分、年纪、场景）

比如:做服装行业的，行业词:牛仔裤，关键词:胖子，输入百度找到大家都在搜，然后看是否有合适的选题，如果有继续点开此选题，继续找到该词条下的大家都在搜，找到适合自己的选题。

抖音:先综合搜索——向右找到话题点击——找到最热的点击，拆解拍的好的人的视频。

巨量算数:输入赛道关键词——点击关联分析——向下找到雷达图，找到关键词，然后可以随意组合关键词找出新的话题，把组合的话题输入搜索框搜索，会有很多视频选择其中自己想拍的去拆解，把适合自己选题装到自己素材库。

热点事件:今日热榜、微博热搜、抖音热榜。蹭热点必须是你的用户群体关心的话题。

呈现方式:同领域找选题，跨领域找呈现。

对标账号:差异化策划，跨行业借鉴。

跨领域、跨平台同领域、跨平台跨领域。

开篇:

减肥:1个人，1……，1……，就能瘦3-5斤

小红书:小红书笔记封面的文字，当成开篇。

比如:我发现了一套人生作弊系统。（我发现了一个抽中隐藏款盲盒的作弊系统）

英语:我发现了一个快速背单词的作弊系统，我发了一套快速增加词汇量的作弊系统，我发现了一套快速雅思提分的作弊系统。

服装:我发现了一个矮个子能穿出一米八气质的穿搭作弊系统。

比如:所有女生六大存钱法，存不下来算我输。

减肥:所有小腿粗的女生六大方法，瘦不下来算我输。

金句:结尾

知乎:搜索鼓舞人心的句子，对应自己的赛道。

爆款选题

八大爆款元素:成本，人群，反差，奇葩，最差，怀旧，头牌，荷尔蒙。

一:选题:吃饭

成本:1000块钱吃一顿外卖/一分钱不花吃到撑。

奇葩:一天只吃黑色的食物。

怀旧:复刻90年代校门口路边摊。

头牌:尹正同款蒸菜减脂餐。

服装赛道:教知识、成本元素。

1.50块钱搭一身潮女服装。100块钱打造一身甜妹装。一件T恤搭配一周不重样。

选题:创业

成本:5000块钱服装创业。

人群:95后裸辞摆摊卖土豆泥。

反差:让外卖小哥管理一天公司。

减肥:教知识、人群元素

人群:天生喝凉水都长胖的，大肚子，大基数小基数，居家的。

二:价值垂直:年轻女性:旅游享受，美食，升职加薪，情商。

内容配比:赛道内容为主，人群内容为辅，比如自己核心变现的内容要占百分之90，其他类选题只能占百分之十。

学会创作

一:情绪波点:

回忆——故事型脚本

行动——过程型脚本。

分析——知识/观点型脚本。

文案写作基础

写文案→说画面。

好的文案就是一个特写镜头。

画面感方法:描述、类比（把一件不具体的事物类比成具象的事情），力量感（强弱对比，对比数量，对比时间，对比经历，对比身份）

说观点脚本

一:写作对象:

单亲妈妈+遗憾=没给孩子一个完整的家，以后他会不会怪我？

恋爱中的女生+纠结=发现男朋友玩聊天软件，该不该原谅他？

单亲妈妈+伤害=用人单位知道我是单亲妈妈，就拒绝录用我。

二:观点型选题创作技巧:人物关系+行业。
