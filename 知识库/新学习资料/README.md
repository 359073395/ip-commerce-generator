## 资料处理方式

- 视频：分段转写，保存原始转写，再整理为可调用方法。
- 文档/表格/PDF/PPT：转换为 Markdown，提炼方法、流程、模板、案例。
- 图片课件/手册：优先提取文字与结构，必要时保留来源路径。
- 示例表格：抽取字段、流程、填表示例，转成可复用模板。

## 输出目标

- `带货视频`：账号定位、选品、成交选题、直播/小黄车、投流、转化脚本、复盘流程。

- 应用流程：以后用户给行业/产品后，可直接生成 IP 规划、内容计划、脚本和执行清单。
