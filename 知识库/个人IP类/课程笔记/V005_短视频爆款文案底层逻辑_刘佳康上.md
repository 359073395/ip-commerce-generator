## 一句话结论

爆款文案的底层逻辑是：先满足平台对停留和合规的要求，再满足用户对共鸣、冲突、好奇和利益的需求。

## 核心框架

### 1. 爆款不是偶然，爆过的文案可以复用

课程开头提出，一个视频爆了之后，把文案翻拍、复拍、换账号再发，仍然可能继续爆，说明爆款背后有稳定逻辑。

可复用洞察：

- 爆款内容不是单纯灵感，而是结构有效。
- 爆过的文案要沉淀进文案库，而不是发完就丢。
- 复用时重点复用结构和情绪，不照搬表达。

### 2. 先满足平台：合规 + 停留

平台需求主要是两点：

- 合规：不让平台承担违规风险。
- 价值：给平台带来用户停留和商业价值。

课程特别强调平均播放时长。复盘视频时，要看用户在哪个时间点流失，并判断那一段讲了什么，下次删掉或改写。

可复用洞察：

- 短视频最重要的数据之一是平均播放时长。
- 五秒完播、两秒跳出、平均播放时长都要看。
- 平台给流量的前提是你能帮它留住用户。

### 3. 再满足用户：共鸣 + 冲突

用户需求主要是两点：

- 共鸣：影响用户当下情绪。
- 冲突：影响用户当下思考和质疑。

课程提醒，创作者不能只站在自己角度判断“我觉得会爆”，而要站在用户角度判断：如果推给 100 个目标用户，有多少人会愿意停下来。

可复用洞察：

- 共鸣让用户觉得“这说的是我”。
- 冲突让用户觉得“这和我想的不一样”。
- 文案要从用户视角做判断，不从作者自嗨出发。

## 流量题材层级

### 第一梯队：最大众、最强停留

1. 生命力：健康、疾病、养生、安全、风险。
2. 钱：赚钱、省钱、亏钱、机会、资产。
3. 性张力：亲密关系、性教育、隐私话题，需合规表达。
4. 好奇心：未知、挑战、危险、悬念、恐惧。

注意：好奇心内容不要在前三秒把答案讲完，要持续制造“我想知道接下来会怎样”。

### 第二梯队：关系议题

- 父母关系。
- 夫妻关系。
- 子女关系。
- 朋友关系。
- 职场关系。
- 人生规划。

这类内容容易产生情绪共鸣，适合个人IP做故事、观点和价值观输出。

### 第三梯队：垂直/变现内容

- 只和小部分人相关的专业内容。
- 和产品、课程、服务直接相关的营销内容。

这类内容天然流量较小。要做爆，需要把第一梯队或第二梯队的流量因子嵌进去，再连接到产品和变现。

可复用洞察：

- 营销内容直接讲产品，流量通常会下降。
- 垂直内容要想爆，要借用更大众的情绪、关系、钱或好奇心。
- “泛题材 + 垂直转化”比纯硬广更适合个人IP。

## 开头钩子原则

课程提到多个常用开头结构：

- 提问/反问。
- 直接列出几条。
- 留悬念。
- 反常识。
- 主题标题直接切入。
- 名人/热点解读。
- 强观点。
- 数字化表达。

其中强观点要清晰，不要含糊。数字越具体越有信服力，例如“一百万/两百万”比“几百万”更容易被用户感知。

## 案例拆解方法

课程用长故事视频做案例时提醒：拆案例不要被故事情节带进去，而要看：

- 每一段讲了什么。
- 段落之间怎么衔接。
- 情绪如何推进。
- 哪些信息负责制造停留。
- 结尾如何连接价值观或产品。

## 可复用结构

```text
平台合规 -> 平均停留 -> 用户共鸣 -> 用户冲突 -> 题材层级 -> 钩子结构 -> 案例拆解 -> 垂直转化
```

## 适合改编的选题

1. 为什么你的视频很专业，但平台不给流量。
2. 爆款文案不是靠金句，而是靠停留。
3. 做营销视频没流量，因为你没加大众情绪入口。
4. 短视频前三秒，千万不要把答案说完。
5. 为什么“几百万”不如“一百万”有力量。

## 风险与注意

- 健康、金钱、性相关题材虽然容易带来停留，但必须注意平台合规和表达边界。
- 不应为了冲突制造虚假信息或夸大恐惧。
- 转写中有大量错字，已按语义归纳。
