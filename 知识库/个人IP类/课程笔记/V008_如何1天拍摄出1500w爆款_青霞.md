## 一句话结论

拍摄爆款不是现场碰运气，而是在开拍前通过调研、对标、场景、文案和IP状态，把胜率提前做出来。

## 核心框架

### 1. 先胜后战：开拍前先确定方向

课程引用“胜兵先胜而后求战”，放到短视频里就是：在拍摄前先确定方向、定位、对标和拍摄形式是对的，再开始拍。

可复用洞察：

- 不要拿拍摄现场去试错方向。
- 拍摄前的调研决定内容胜率。
- 如果没找到对标，说明还没真正完成调研。

### 2. 对标调研是最大的捷径

青霞用亲子户外/拉练案例说明：即使没有完全一样的同行，也可以拆产品中的某个部分，找相似人群、相似场景、相似需求的对标。

对标路径：

- 找同行产品。
- 找相似产品。
- 找相似年龄段/目标家庭。
- 拆分产品步骤，找局部相似场景。
- 找评论区有购买意向的账号。

可复用洞察：

- 好对标不一定行业完全一样，但用户动机要相似。
- 评论区要有“怎么报名、多少钱、怎么参加”等转化信号。
- 调研能节省别人几百条视频试错的成本。

### 3. 拆对标要拆画面和文案目的

课程把对标视频拆成两部分：

- 画面：每个镜头多长、画面里是什么、如何切换。
- 文案：每一句在完成什么任务，比如人设、痛点、期待、转化。

可复用洞察：

- 不要只看别人说了什么，要看每一句为什么存在。
- 先拆“目的”，再换成自己的表达。
- 画面和文案要同步服务同一个转化目标。

### 4. IP状态比文案本身更重要

同一篇爆款文案，不同IP拍出来结果可能完全不同。因为每个IP都有自己适合的表达方式、情绪状态和角色感。

可复用洞察：

- 文案要和IP本人匹配。
- 有的人适合讲故事，有的人适合讲干货，有的人适合强情绪。
- 不要把不适合某个IP的爆款硬套给他。

### 5. 找IP最佳状态的三个入口

1. 面谈本人：了解生活、工作、性格和表达习惯。
2. 访谈家人/团队：了解他平时喜欢说什么、什么情境下有情绪波动。
3. 观察工作状态：他在什么时间、地点、场景下输出最好。

可复用洞察：

- 把IP线下最自然的状态搬到镜头前。
- 美业/微商老板可能适合会销式讲课场景。
- 心理/咨询类IP可能更适合安静坐下来深度表达。

### 6. 特点比完美更重要

课程用“老师骂人”的案例说明：一个特点从旁观者看未必完美，但如果它能给用户留下强烈第一印象，并且用户后来认可“骂得对”，它就可能成为账号记忆点。

可复用洞察：

- 短视频不怕有争议特点，怕没有特点。
- 用户记住你，比你看起来完全正确更重要。
- 真实特点要服务用户价值，不能只是情绪宣泄。

### 7. 表现力靠刻意练习

课程用雷军早期视频和花絮说明，没有人天生镜头表现力完美，表现力是练出来的。

可复用洞察：

- 第一次拍不自然是正常的。
- 训练要复盘自己的视频，而不是只重复拍。
- 创始人/老板要放下面子，用户连接比形象包袱更重要。

## 可复用结构

```text
调研对标 -> 拆画面 -> 拆文案目的 -> 匹配IP状态 -> 设计拍摄场景 -> 强化特点 -> 复盘表现力
```

## 适合改编的选题

1. 为什么同一篇爆款文案，别人拍爆了你没爆。
2. 拍摄前不做调研，就是拿时间和钱试错。
3. 对标视频要拆的不是台词，而是每一句的目的。
4. 个人IP最怕不是有缺点，而是没有记忆点。
5. 老板拍视频不自然，根本原因不是镜头，而是状态没找对。

## 风险与注意

- “抄对标”应理解为拆结构和表达目的，不是搬运画面与文案。
- 有争议的人设特点要确保不伤害用户、不违反平台规则。
