# 个人IP类知识库

个人IP类内容的核心目标是让用户从“看见你”走到“理解你、信任你、记住你”。

## 重点观察

- 账号想让用户记住的身份是什么。
- 内容是否有稳定的人设信号：职业、经历、性格、审美、价值观。
- 观点是否清晰，有没有冲突和记忆点。
- 价值输出是否具体，能不能让用户立刻获得启发。
- 结尾是否给出关注理由或持续关系。

## 常见内容类型

- 观点表达
- 经验复盘
- 故事分享
- 知识教学
- 行业洞察
- 日常记录
- 问答回应
- 个人成长

## 口径说明

本文件夹不是独立于带货的“纯个人IP类”，而是“个人IP带货”中的信任与内容视角。

这里重点观察：

- IP如何定位。
- 人设、观点、故事和结果如何建立信任。
- 内容如何带来关注、进主页、进直播和进私域。
