# 直播运营方法论

来源：V007-V009《25年直播运营核心维度拆解》美晞。

## 总公式

```text
人群定位 -> 流量入口 -> 5分钟话术 -> 互动停留 -> 货盘分层 -> 活动排期 -> 数据复盘 -> 付费放大
```

## 四类货盘

| 类型 | 作用 | 注意 |
|---|---|---|
| 福利款 | 拉人气、制造限时感 | 不负责利润 |
| 背书款 | 建立品牌和信任 | 可以低利润 |
| 盈利款 | 承担主要利润 | 需要清晰卖点 |
| 平播款 | 稳定讲解和承接 | 随时能讲顺 |

## 5分钟话术模块

- 我是谁/直播间在干什么。
- 当前福利或主题。
- 二选一互动。
- 发散性互动。
- 专业内容或产品卖点。
- 点赞、关注、粉丝群、下单等运营动作。
- 下一个 5 分钟循环重新开始。

## 复盘指标

- 曝光。
- 进入率。
- 停留。
- 互动。
- 关注。
- 转粉。
- 点击。
- 下单。
- 客单。
- 转化率。

## 新手原则

- 先自然流，不要急着投流。
- 先一个入口，不要公域、短视频、私域同时抓。
- 先开播，再优化，不要等全部准备完。
- 先测货，再放大。
- 先复盘，再拉长时长。
