# 素人 IP 起盘方法论

## 核心判断

素人 IP 起盘的第一目标不是马上精准变现，而是先让账号活起来，再把流量、信任、直播、私域和变现按阶段串起来。

## 四阶段模型

### 1. 账号启动期

目标：找到能爆的内容形式。

动作：

- 明确 IP 和项目价值。
- 做行业、人群、内容、拍摄形式对标。
- 泛流量内容占比更高。
- 先做秀结果、人设故事、互动情绪类内容。

### 2. 引流优化期

目标：把流量变成可承接的人。

动作：

- 优化简介和置顶视频。
- 根据爆款内容开直播承接。
- 测试直播间引私域话术。
- 复盘哪些内容带来有效客户。

### 3. 直播转化期

目标：让用户在直播间建立信任。

动作：

- 用短视频给直播间导流。
- 在直播间讲清 IP 价值和用户问题。
- 用直播素材继续二次分发。
- 逐步搭建私域或销售承接。

### 4. 稳定放大期

目标：从单账号变成整盘运营。

动作：

- 调整泛内容和垂直内容比例。
- 建立内容矩阵和直播节奏。
- 建立私域、销售、交付 SOP。
- 设计更多变现路径。

## 内容比例参考

```text
1.0 阶段：泛内容 70% / 垂直内容 30%
2.0 阶段：泛内容 50% / 垂直内容 50%
变现阶段：泛内容 30% / 垂直内容 70%
```

## 对标方法

不要只找同行。按下面四类找：

- 内容对标：什么表达形式容易爆。
- 行业对标：同行如何变现。
- 人群对标：同一批用户还看什么账号。
- 拍摄对标：同类情绪、场景、镜头如何呈现。

## 三类基础内容

- 秀结果：证明“我做成过”。
- 人设故事：证明“我是谁，为什么可信”。
- 互动情绪：负责让账号活跃，制造转发和评论。

## 一句话复盘

素人 IP 起盘要先解决“有没有人看”，再解决“是不是精准”，最后解决“能不能稳定变现”。
