# AI短视频生产线

来源：V010-V011《AI短视频核爆力搭建 IP 自动化生产线》鲜辉。

## 总公式

```text
项目成立 -> 建爆款库 -> AI改写文案 -> 数字人生成画面 -> 单平台测试 -> 数据反馈 -> 矩阵复制
```

## 三个内容库

| 内容库 | 作用 | 判断标准 |
|---|---|---|
| 流量文案库 | 拿播放、曝光、涨粉 | 播放高、互动高、题材有大众入口 |
| 客资文案库 | 拿咨询、私信、加微信 | 评论区有“怎么买/怎么学/怎么加入” |
| 人设故事库 | 建信任和记忆点 | 有转折、痛点、结果、价值观 |

## 两个核心工具方向

- AI文案：DeepSeek/火山引擎等，用于脚本、开头、标题、改写、违规处理。
- 数字人画面：飞影数字人等，用于批量生成口播素材。

## 数字人素材原则

- 正视镜头。
- 动作小。
- 声音清晰。
- 普通话尽量标准。
- 多录服装和场景。
- 先投市场，再迭代。

## 组织原则

- 先集中打穿一个平台。
- 先跑出流量正反馈，再跑客资正反馈，再跑变现正反馈。
- 一个内容负责人管理约 7 人以内。
- 普通员工只做清晰切分的执行动作。
