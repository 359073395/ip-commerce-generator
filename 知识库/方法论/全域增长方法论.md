# 全域增长方法论

## 核心判断

全域增长不是把所有平台都做一遍，而是把“获流、承接、转化、复购、交付、放大”连成一个系统。

## 基础模型

```text
多平台获流 -> 私域承接 -> 社群教育 -> 一对一顾问 -> 课程/体验品 -> 产品二转 -> 深度服务三转 -> 老客复购 -> 矩阵放大
```

## 关键原则

### 1. 平台分散，客户沉淀

平台用于获流，客户资产要沉淀到可持续触达的体系里。不要把账号当作唯一资产。

### 2. 前端低门槛，后端高价值

前端产品或课程降低进入门槛，后端通过持续服务、专业交付和复购机制提高客户价值。

### 3. 内容负责认知，社群负责教育，顾问负责信任

尤其在高客单、健康、美业、教育、咨询等行业，单靠公开内容很难完成深度成交。

### 4. 交付也是转化

训练营、直播课、线下活动、服务过程都不是单纯成本，而是二转三转和复购的基础。

### 5. 先测通，再复制

任何增长模型都应先小规模验证转化率、交付成本、售后压力和复购，再进行主播矩阵、渠道合作或招商放大。

## 常见误区

- 只盯平台流量，不设计后端承接。
- 只卖单品利润，不做复利结构。
- 没跑通模型就招商复制。
- 敏感赛道忽视合规和平台风险。
- 客户数据、分账和归属没有提前设计。

## 一句话复盘

全域增长的核心，是把不稳定的平台流量，转化为稳定的客户关系和后端复购体系。
