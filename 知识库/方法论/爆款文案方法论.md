# 爆款文案方法论

来源：V005-V006《短视频爆款文案的底层逻辑》刘佳康。

## 总公式

```text
合规 -> 停留 -> 共鸣/冲突 -> 题材层级 -> 钩子 -> 节奏 -> 主线 -> 转化
```

## 核心判断

- 平台要的是合规和停留。
- 用户要的是共鸣、冲突、好奇和利益。
- 营销内容天然流量低，需要嵌入大众题材入口。
- 长故事必须有情绪曲线和主线反复。
- 抄爆款要抄结构，换成自己的案例和表达。

## 文案自检

| 问题 | 判断 |
|---|---|
| 前三秒是否有钩子 | 用户是否必须继续看才知道答案 |
| 是否合规 | 是否有夸大、敏感、风险表达 |
| 是否有大众入口 | 生命、钱、关系、好奇、冲突 |
| 是否有主线 | 中间是否反复呼应同一个主题 |
| 是否有新信息 | 每 10-20 秒是否有新点 |
| 是否可转化 | 结尾是否能自然连接产品或私域 |

## 常用钩子

- 提问/反问。
- 数字清单。
- 留悬念。
- 反常识。
- 名人/热点。
- 强观点。
- 具体数字。
- 关系冲突。

```text
我的目标人群：
他们的核心痛点：
我的产品/服务：
我想生成的主题：
请按短视频爆款逻辑输出：
1. 前三秒钩子
2. 核心观点
3. 案例/解决方案
4. 引流指令
要求：口语化、简洁、有冲突、有共鸣，关联我的业务和人设。
```
