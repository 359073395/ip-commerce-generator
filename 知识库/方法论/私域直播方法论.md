# 私域直播方法论

来源：V012《单场百万的私域直播运营》志浩。

## 总公式

```text
客户池 -> 调研 -> 主题 -> 产品 -> 预热 -> 直播 -> 追单 -> 复盘
```

## 直播前

- 做客户调研。
- 选客户最想听的主题。
- 匹配产品和福利。
- 发朋友圈预热。
- 建社群并设置规则。
- 发小课/视频预热。

## 直播中

- 开头讲清楚我是谁、今天解决什么问题。
- 按节奏上核心内容和产品。
- 设置评论互动和抽奖。
- 社群同步核心卖点和痛点。
- 最高在线时推核心产品。

## 直播后

- 截图评论区客户问题。
- 销售一对一回访。
- 统计产品销量和转化。
- 看回放和逐字稿。
- 沉淀好话术、提单话术和下次优化点。
