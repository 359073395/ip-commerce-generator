# 个人IP带货类知识库

## 定位

这套课程统一归类为“个人IP带货 / 个人IP商业化”，不是把“带货”和“个人IP”拆成两个互斥类别。

更准确的理解是：

```text
个人IP是信任入口
内容是流量入口
直播/私域是承接入口
产品/课程/服务是变现入口
交付/复购是长期利润入口
```

## 两个分析视角

现有文件夹先保留，但含义调整为分析维度：

- `个人IP类/`：重点看人设、定位、内容、信任、账号起盘、IP资产。
- `带货类/`：重点看直播、私域、产品、成交、复购、服务标准化。

所有课程都服务同一个目标：用个人IP建立信任，再通过内容、直播、私域和产品服务完成商业化。

## 核心模块

1. IP定位与人设：解决“我是谁、凭什么信我”。
2. 内容获流：解决“用户为什么停留、关注、进主页”。
3. 爆款文案与拍摄：解决“内容如何更容易被传播”。
4. 直播转化：解决“用户如何从看见走向互动和下单”。
5. 私域承接：解决“流量如何沉淀为客户资产”。
6. 产品与服务设计：解决“卖什么、怎么卖、怎么交付”。
7. 复购与全域增长：解决“如何从一次成交变成长期生意”。
8. AI生产线：解决“如何规模化内容生产和运营效率”。

## 使用原则

- 不再问“这是带货还是个人IP”，而是问“这节课在个人IP带货链路里解决哪一环”。
- 一篇笔记可以同时带有 `#个人IP`、`#带货`、`#私域`、`#直播` 等标签。
- 复用时优先拆底层机制，不复制讲师原话、案例表达和具体脚本。
