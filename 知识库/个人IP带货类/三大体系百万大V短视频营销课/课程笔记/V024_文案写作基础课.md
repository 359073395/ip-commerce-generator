## 一句话结论

短视频文案不是写漂亮作文，而是把脑海里的画面说出来。好的文案由两部分组成：画面感和力量感。画面感靠描述法和类比法，力量感靠方言词汇和强弱对比；少用空泛形容词，多用大白话、细节和具体画面。

## 文案不是“写文案”，而是“说画面”

课程反复强调：

```text
写文案 = 说画面
```

很多人写文案时追求成语、金句、漂亮词，结果用户听不进去。

真正吸引人的文案，是用户一听就能在脑海里出现画面。

差：

```text
我很努力创业。
今天天气真好。
她现在变漂亮了。
我当时很危险。
```

好：

```text
我那个彩票摊，三个月赚了一百多万。
天蓝得一点云都没有，偶尔有几只鸟飞过去。
车门打开，一双踩着高跟鞋的长腿落在旧石板路上。
凌晨一点，火车站外没有出租车，我只能一个人站在路边等。
```

## 好文案的两个组成

```text
画面感 + 力量感
```

画面感让用户看得见。

力量感让用户听得进、记得住、感受到情绪。

## 画面感：把形容词落到细节点

空泛形容词很难打动用户。

比如：

- 很努力。
- 很惨。
- 很漂亮。
- 很危险。
- 很大。
- 很厉害。
- 很有钱。

这些词都太虚，用户感受不到。

要把它们落到细节点上。

```text
很惨
-> 三天没洗头，白衣服已经穿成了灰黑色。
```

```text
很大
-> 大到能在家里踢足球。
```

```text
很努力
-> 三年换了三份工作，晚上下班还去摆摊。
```

## 方法1：描述法

描述法就是把一个抽象状态，用具体细节描述出来。

公式：

```text
抽象词
-> 看到什么、听到什么、发生了什么
-> 具体画面
```

例子1：

```text
今天风很大
-> 我站在那儿都不用动，风直接把帽子扣到我头上。
```

例子2：

```text
我一个人在火车站很危险
-> 凌晨一点，车站外一辆出租车都没有，我只能站在路边等。
```

例子3：

```text
我很生气
-> 我转头看见她把我最喜欢的口红拧断了，口红芯直接掉在地上。
```

描述法的关键不是写得华丽，而是具体。

## 方法2：类比法

类比法就是把不具象的东西，比成一个大家熟悉的具象东西。

公式：

```text
抽象/不具象
-> 找一个用户熟悉的参照物
-> 让用户瞬间感受到程度
```

例子1：

```text
我减肥成功了
-> 我三个月瘦了25斤，相当于一桶桶装水。
```

例子2：

```text
工程车特别大
-> 它前面两个轮子跟我差不多高。
```

例子3：

```text
服务很好
-> 服务细到像海底捞一样。
```

类比法适合解释：

- 大小。
- 重量。
- 速度。
- 价格。
- 智商。
- 难度。
- 差距。
- 服务程度。

## 文案要少用四字成语

课程提醒：很多人一写文案就喜欢用四字成语和书面词。

问题是：

- 太官方。
- 没画面。
- 不像真人说话。
- 用户不容易共鸣。

短视频不是作文比赛，用户更喜欢听得懂、看得见、有情绪的大白话。

差：

```text
我当时一无所有，内心迷茫。
```

好：

```text
我把结婚时买的三金都卖了，卡里只剩400块。
```

## 力量感：让文案有冲击

除了画面感，好文案还要有力量感。

力量感来自：

- 更有冲击的动词。
- 方言词汇。
- 强弱对比。
- 情绪更真实的表达。

比如：

```text
我拍你
```

和：

```text
我抽你
```

冲击力明显不同。

动词越准确，情绪越强。

## 方法3：方言词汇

课程特别强调：方言，尤其是东北话，天然有力量感和网感。

因为它：

- 更像真人说话。
- 情绪更直接。
- 画面更强。
- 更容易制造喜感和冲击。

例如一些带口语感、方言感的词，会比普通话更有劲。

但注意：方言不是硬装，适合自己就用，不适合不要强行模仿。

原则：

```text
能让情绪更真实，就用。
让表达变假，就不用。
```

## 方法4：强弱对比

强弱对比可以让用户瞬间感受到差距。

常见对比维度：

- 数量对比。
- 时间对比。
- 经历对比。
- 身份对比。
- 金额对比。

### 数量对比

```text
小王一顿吃1个馒头，我一顿能吃50个。
```

用户立刻感受到差距。

### 时间对比

```text
别人三年做到的事，我一年做到了。
```

### 经历对比

```text
三年里，我干过服务员、销售、讲师，换了三份工作。
```

### 身份对比

```text
我一个新手小白，一年做出了别人团队三年才做出来的东西。
```

强弱对比不是为了夸张，而是为了让用户更快理解程度。

## 文案里的情绪和共鸣

课程指出，现在用户越来越喜欢有情绪、有共鸣的内容。

干货有价值，但情绪价值更容易让用户停下来。

情绪和共鸣从哪里来？

```text
从具体细节里来。
```

当你把“我很难受”说成具体画面，用户才有机会共情。

## 个人IP带货怎么用

带货文案也不要只写参数。

差：

```text
这个锅很好用，质量很好。
```

好：

```text
我用它煎鸡蛋，锅铲一推，蛋边直接滑起来，底下一点都没粘。
```

差：

```text
这个包容量很大。
```

好：

```text
手机、粉饼、钥匙、充电宝、纸巾全塞进去，还能再放一瓶水。
```

差：

```text
这个清洁剂去污力强。
```

好：

```text
灶台上那块发黄的油印，我喷完等了十秒，纸一擦就下来了。
```

用户买的是商品，但被打动的是画面。

## 文案写作公式

```text
先写出抽象句
-> 找到里面的形容词
-> 问自己：具体表现在哪里？
-> 用描述法写细节
-> 用类比法解释程度
-> 加入强弱对比
-> 用更像真人说话的表达
```

## 练习题

把下面句子改成画面：

```text
我很累。
```

可改：

```text
我坐在楼道台阶上，外卖袋还拎在手里，鞋都没脱就睡着了。
```

```text
这个东西很划算。
```

可改：

```text
一盒能用三个月，折下来一天不到一块钱。
```

```text
我家服务很好。
```

可改：

```text
你下单后尺码拿不准，客服会把你的身高体重和试穿图一张张对给你看。
```

## 文案检查表

```text
1. 有没有空泛形容词？
2. 每个形容词有没有落到具体细节？
3. 用户听完能不能脑补画面？
4. 有没有用熟悉事物做类比？
5. 有没有数量、时间、经历或身份对比？
6. 表达是不是像真人说话？
7. 有没有过度使用四字成语？
8. 带货文案有没有具体使用画面？
```

## 行动清单

- 把最近写过的10句文案里所有形容词圈出来。
- 每个形容词改成一个具体画面。
- 每条带货文案至少补一个使用细节。
- 每条口播至少加一个类比或强弱对比。
- 用大白话重写所有太官方的句子。
- 拍摄前把文案读一遍，检查像不像自己平时说话。
