## 一句话结论

拆片不是抄文案，而是把别人的爆款视频拆成“账号层逻辑”和“单条视频逻辑”，看清它为什么有人看、为什么能让人看完、为什么适合这个人拍，然后把结构迁移到自己的行业、人设和选题里。

## 为什么要学拆片

前面课程已经讲了观点、晒过程、教知识、讲故事等基础脚本，但基础脚本不能支撑账号一辈子。短视频创作没有固定终点，真正持续进步的方法，是长期学习优秀创作者的表达结构。

课程里的核心判断：

- 创意大多来自看见、分析和改造，而不是凭空出现。
- 你看得多、拆得多，才能不断积累新选题、新结构、新表达。
- 拆片训练的目标是提升网感和创作能力，不是照搬别人。

## 拆片总框架

课程把拆片分成两大部分：

```text
账号策划部分
-> 内容类型
-> 主题系列
-> 人设类型

单条内容部分
-> 写给谁
-> 爆款元素
-> 脚本结构
```

先拆账号，再拆单条。只拆单条，很容易学到皮毛；先看账号定位和人设，才知道这条视频为什么适合这个人拍。

## 一、账号策划部分

### 1. 内容类型

先判断这条视频属于什么内容方向：

- 观点。
- 教知识。
- 晒过程。
- 讲故事。
- 广告。
- 新闻。
- 剧情。

学完前面的脚本课后，刷视频要带着“公共降噪”的视角：不是只觉得好看，而是能识别它属于哪种内容类型。

### 2. 主题系列

主题系列指成体系、可连续更新的选题。例如：

```text
用AI设计人类从没见过的料理
-> 可以迁移为：用AI设计人类从没见过的家具/门店/包装/穿搭
```

再比如：

```text
那些不显眼但很赚钱的账号
-> 可以迁移为：那些不显眼但很赚钱的门店/产品/服务/岗位
```

拆主题系列时要做两步：

- 提取原选题的通用结构。
- 替换成自己的行业和对象。

主题系列的价值是能让你一次性得到一组内容，而不是只得到一条视频。

### 3. 人设类型

课程总结了抖音上常见的人设类型。重点不是选最厉害的人设，而是选最真实的人设。

### 崇拜者人设

用户喜欢你、欣赏你、佩服你，所以愿意跟你买。它靠人格魅力和经历吸引人，不一定靠专业。

适合：

- 做了别人做不到的事。
- 有强烈个人魅力。
- 明星、运动员、公众人物、极限挑战型博主。

### 教导者人设

用户因为你专业而信你。你看起来是某个领域从业多年的人、专家、老师、医生、顾问、老板。

适合：

- 有行业经验。
- 有专业身份。
- 有大量案例和方法。
- 能给用户明确建议。

### 分享者人设

你不是专家，但你在某件事上做得不错，所以把经验分享给别人。

适合：

- 身材管理做得好，但不是营养师。
- 口腔护理做得好，但不是医生。
- 家庭收纳做得好，但不是职业整理师。
- 穿搭、护肤、学习、生活方式经验分享。

### 陪伴者人设

你还没有得到最终结果，但正在解决问题的路上，把过程记录下来。也可以理解为成长型人设。

适合：

- 新手入行。
- 减肥、变美、创业、学习、恢复、改造过程。
- 和用户一起面对同类问题。

### 衬托者人设

主动展示自己的弱点、短板、不如人的地方，让用户产生关注、好奇、共情或保护欲。

可能来自：

- 身体健康。
- 经济基础。
- 学历。
- 职业处境。
- 家庭背景。
- 特殊处境。

### 搞笑者人设

以段子、短剧、夸张表演、反差制造娱乐价值。

### 人设选择原则

不要问“哪个人设最赚钱”，要问“哪个人设最真实”。你生活里是什么样的人，账号里就应该是什么样的人。

如果你有二十年经验，就做教导者；如果你刚入行，就做陪伴者；如果你只是把某件事做得不错，就做分享者。硬装富二代、专家、精英，用户会觉得不真实，内容也很难成立。

## 二、单条内容部分

### 1. 写给谁

一条视频必须有明确对象。不是写给所有人，而是写给一个具体人群。

拆片时要问：

- 这条视频在替谁说话？
- 它解决谁的焦虑？
- 它吸引谁继续看？
- 这个人群的阶段、身份、痛点是什么？

例如“孩子该不该报兴趣班”的视频，写作对象不是所有家长，而是“正在纠结要不要给孩子报兴趣班、并且被别人质疑的家长”。

### 2. 爆款元素

爆款元素是视频里让人愿意点开、停留、继续看的东西。

常见爆款元素包括：

- 反常识。
- 低成本。
- 高收益。
- 强冲突。
- 高难挑战。
- 好奇心。
- 反差。
- 隐秘空间。
- 明确结果。
- 让用户做梦。

例如减肥知识视频里，“不用节食、不用运动也能瘦”打的是低成本；探访美女家的过程视频里，“单身女网红的家、卧室、秘密”打的是好奇心和暧昧空间；挑战类视频里，“高难任务+自信接受+过程翻车”打的是反差和看点。

### 3. 脚本结构

脚本结构是拆片最关键的部分。方法是：

1. 把完整文案分成自然段。
2. 看每一段在做什么。
3. 去掉具体人物、行业、事件这些“肉”。
4. 留下表达逻辑这副“骨架”。
5. 用自己的行业、人设、选题重新填肉。

拆片时不要纠结你总结出来的词是否和老师一模一样。用自己的话总结也可以，关键是你能看懂每一段的作用，并能迁移使用。

## 案例结构一：观点视频

拆片逻辑：

```text
我被某类人质疑/劝说
-> 我先承认或描述对方的逻辑
-> 我举出某个正面例子，说明我的选择有价值
-> 我指出对方那套逻辑的问题
-> 我表达自己的观点，并说明以后还会坚持这么做
```

可迁移场景：

- 有人说我不会来事。
- 有人说我不懂社交。
- 有人说我不该给孩子报兴趣班。
- 有人说我做生意太较真。
- 有人说我拍视频太真实、不够包装。

这类视频的价值在于：把被质疑的经历转成观点表达，让目标用户觉得“你说出了我的心声”。

## 案例结构二：知识视频

以减肥方法类内容为例，它并不是一上来就教方法，而是先让用户相信“这个方法适合我”。

结构：

```text
我在某个领域取得了结果
-> 我原来基础也不好，和你一样
-> 高难度方法我也做不到
-> 我的方法很低成本、很好坚持
-> 用它能得到你想要的结果
-> 给出具体操作步骤
-> 结尾催行动
```

关键点：

- 先证明你做到了。
- 再证明你原来也不行。
- 再降低行动成本。
- 再让用户看到理想结果。
- 最后才教方法。

这比直接讲步骤更容易让用户停下来听。

## 案例结构三：晒过程视频

探访类、挑战类、体验类视频常用。

结构：

```text
高能片段预告
-> 公布今天的主线任务
-> 加入一个吸引人的爆款元素
-> 路上持续透露线索
-> 到达目标场景
-> 进入用户最想看的空间/环节
-> 加入小冲突或小秘密
-> 展示最终结果
```

这类视频的关键是期待感。过程本身可能很简单，但每一步都要有一个让用户继续看的理由。

## 案例结构四：高难挑战视频

挑战类视频可以拆成：

```text
提出高难任务
-> 人物信心满满接受
-> 先小试一下，发现比想象中难
-> 人物想逃跑或后悔
-> 被旁人拉回来，不能丢脸
-> 开始耍赖、讲条件、找理由
-> 最后认真完成挑战
-> 公布结果
-> 用痛苦反应制造反差收尾
```

可迁移到：

- 请朋友吃饭挑战。
- 门店高客单服务挑战。
- 一天完成某个高难任务。
- 预算内改造挑战。
- 体验某个极端场景。

挑战类最重要的不是完成，而是过程里的反差、逃避、被拉回、耍赖和最后结果。

## AI辅助拆片

AI可以提高拆片效率，但前提是你自己已经会基本拆片。

### 工具流程

```text
视频转文字
-> 把文案发给AI
-> 让AI先分自然段
-> 让AI逐段做结构拆解
-> 让AI给出仿写公式
-> 人再判断是否可用
```

### 提示词关键

不要只让AI“拆解结构”。否则它可能给你一堆抽象词，例如“现实坠落”“精神反弹”之类，看起来高级，但不知道怎么写。

更好的要求是：

```text
请把这条视频文案分成自然段。
逐段拆解每一段的作用。
每一段都给我一个可直接仿写的公式。
公式要让我看完就知道这一段该写什么。
```

AI适合帮你提速，但不适合替你建立网感。课程建议：先手动拆几十条视频，再用AI提效。

## AI不适合拆什么

### 1. 高度依赖视觉的视频

例如美食特写、精致画面、食材质感。AI只看文案，很难理解画面为什么让人停留。

### 2. 高度依赖手艺的视频

例如修复、打磨、雕刻、改造、手工制作。视频能火可能是因为手艺本身厉害，不是脚本结构厉害。

### 3. 高度依赖情绪表演的视频

例如情感短剧、小短片、演员表情、音乐、构图、镜头调度。只给AI文案，它看不到动作、表情、节奏和音乐。

## 不建议拆的爆款

### 1. 广告和挂车视频

挂车视频、投流视频的高数据可能来自投放，不一定来自内容本身。除非你也投钱，否则拆它的自然流量逻辑意义有限。

### 2. 特殊身份视频

明星、公众人物、特殊职业、特殊身体状态等，流量可能来自身份，而不是内容结构。普通账号直接学，容易学错。

### 3. 没有商业目的的视频

如果你的目标是个人IP带货、获客、商业化，就不要过度拆擦边、猎奇、纯娱乐、负面挑逗类内容。它们可能有流量，但不一定服务人设和成交。

## 拆片表格模板

```text
账号名：
行业/赛道：

账号策划部分：
1. 内容类型：
2. 主题系列：
3. 人设类型：
4. 我能迁移的系列选题：

单条内容部分：
1. 视频标题/主题：
2. 写给谁：
3. 爆款元素：
4. 分段结构：
   - 第1段：
   - 第2段：
   - 第3段：
   - 第4段：
5. 去肉留骨后的脚本公式：
6. 我自己的行业替换：
7. 可拍成什么视频：
8. 风险点：
```

## 行动清单

- 每天拆3条同赛道或邻近赛道视频。
- 每条视频先判断内容类型，不要先抄标题。
- 每周整理3个可复用主题系列。
- 每条视频都标注写作对象和爆款元素。
- 每条视频至少拆出一个“去肉留骨”的脚本公式。
- 先手动拆片建立网感，再用AI提高效率。
- 避开投流广告、特殊身份和无商业目的的内容。
