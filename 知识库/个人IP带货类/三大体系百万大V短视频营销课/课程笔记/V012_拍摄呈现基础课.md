## 一句话结论

短视频呈现不是“拿起手机随便拍”，而是要先选对呈现形式，再根据内容选择口播视角或过程视角，最后用景别、构图、运镜把画面做得更清晰、更自然、更有IP感。

## 四种常见呈现形式

### 1. 图文

特点：

- 图片加背景音乐。
- 成本很低。
- 人设弱。
- 现在自然流量和变现能力都比较弱。

适合做补充，不适合作为个人IP带货账号的主力形式。

### 2. 情景剧/短剧

特点：

- 流量可能很大。
- 用户爱看。
- 拍摄成本高。
- 需要演员、剧本、场地、剪辑。
- 角色在演别人，人设和IP沉淀较弱。

适合团队化内容，不适合大多数普通个人IP初期直接上手。

### 3. 口播

特点：

- 人设强。
- 变现好。
- 成本低。
- 一个人也能拍。
- 对表现力、文案、场景、拍摄状态有要求。

个人IP带货最优先学习的呈现方式之一。

### 4. Vlog/过程

特点：

- 人设强。
- 真实感强。
- 变现好。
- 适合晒过程、测评、体验、挑战。
- 拍摄和剪辑成本比口播高。

课程建议：优先掌握口播和Vlog/过程，这两种最适合个人IP商业化。

## 口播的四种视角

### 1. 自拍视角

自己拿手机或固定机位对着自己说。

优点：

- 成本最低。
- 一个人就能拍。
- 适合情绪表达、个人分享、观点输出。

缺点：

- 对表现力要求高。
- 背景和状态不好时容易显得单调。

建议：不要站在白墙前硬拍，尽量找有纵深、有生活信息的背景。

### 2. 偷拍视角

像是被别人无意拍到，或像第三方记录现场。

优点：

- 真实感强。
- 用户容易有“吃瓜感”。
- 适合街访、现场冲突、真实交流、服务过程。

适合：想弱化表演感、增强现场感的内容。

### 3. 聊天视角

两个人自然聊天，一个人拍另一个人，或两个人在真实关系里交流。

关键点：

- 一定要有真实的人物关系。
- 关系越亲密，状态越自然。
- 适合夫妻、姐妹、朋友、老板员工、老师学生等关系。

聊天视角能让IP状态更松弛，用户更容易听进去。

### 4. 采访视角

一个人问，一个人答；也可以双机位拍摄，剪辑时切换。

优点：

- 不需要完整背稿。
- 只要提前列问题。
- 被采访者回答更自然。
- 适合专家、老板、客户案例、门店服务、员工故事。

## Vlog/过程的呈现方式

过程类视频可以从弱到强分为几类：

- 只有画面，没有人出镜、没人说话：有看点，但IP弱。
- 画面配旁白：能讲清楚，但人物连接感弱。
- 真人说话但不完整出镜：有信息，也有一定人设。
- 真人出镜 + 真人说话 + 完整过程：IP感和变现能力最好。

课程强调：个人IP带货最推荐的过程呈现，是“真人出镜并讲清楚过程”。

## 过程视频怎么拍

过程视频要“正常拍”，也就是把事情真实发生的流程拍出来。

例如“下班回家吃饭”可以拆成：

```text
下班：收拾工位 -> 打卡 -> 出门
回家：打车/开车 -> 路上
吃饭：买菜 -> 到家 -> 做饭 -> 吃饭
```

不要只拍一个结果，要拍每个环节里应该发生的信息。用户看过程，就是想看“事情怎么一步一步发生”。

## 画面质量的四个维度

课程把画面基础拆成：

```text
置景 -> 景别 -> 构图 -> 运镜
```

设备会影响质量，但在基础阶段，先把这四件事做好，比一味换设备更重要。

## 一、景别

景别是镜头里“人和环境的距离关系”。

### 1. 远景

人小，环境大。主要交代环境。

适合：

- 开场交代地点。
- 展示空间规模。
- 表达孤独、忙碌、宏观氛围。

### 2. 全景

人物全身和环境都在画面里，以人物为主体。

适合：

- 展示人物和环境的关系。
- 展示完整动作。
- 展示人与物的空间关系。

### 3. 中景

通常拍到膝盖、腰部或上半身，重点看人物动作。

适合：

- 过程动作。
- 讲解操作。
- 人物正在做某件事。

### 4. 近景

胸口以上或人物脸部更清楚，重点是状态、表情和情绪。

适合：

- 口播。
- 情绪表达。
- 观点输出。
- 强化人设。

### 5. 特写

局部细节，比如眼泪、手、产品细节、某个动作。

适合：

- 强化情绪。
- 强化关键细节。
- 展示产品质感。

### 景别口诀

```text
人小景大是远景
人全常用是全景
动作讲解用中景
情绪人设用近景
局部放大是特写
```

## 景别使用原则

### 1. 少用远景，多用近景

短视频突出的是IP和人设，不是风景。远景可以用来交代环境，但不要长时间使用。口播和个人IP内容要多用中景、近景。

### 2. 切镜要有明显但不突兀的变化

相邻景别变化太小，用户感受不到切换；跳得太大，又会显得突兀。更好的方式是隔一个景别切换，让画面有变化但不吓人。

### 3. 不要卡关节

画面边缘不要刚好切在手腕、手肘、膝盖、脚踝等关节位置，会显得别扭。

## 二、构图

构图决定用户第一眼看哪里，也决定画面是否舒服。

### 1. 中心构图

主体在画面正中间。

适合：

- 单人口播。
- 强化IP主体。
- 让用户注意力集中到人身上。

错误示例：明明人在讲话，但镜头中间空着，用户会不知道该看谁。

### 2. 三分构图

把画面分成三等份，人物放在三分之一处，并给视线方向留空间。

适合：

- 对话。
- 侧身交流。
- 更有呼吸感的画面。

注意：人物看向哪里，哪里就要留空间。不要让眼神方向贴边。

### 3. 前景式构图

镜头前放一个虚化物体，让画面更有层次，主体更突出。

适合：

- 情感类内容。
- 氛围感内容。
- 深度表达。
- 想让用户更进入画面的内容。

前景不要抢主体，只负责制造层次和氛围。

## 常见构图错误

- 眼神方向没有留白。
- 头顶贴着画面边缘。
- 上方空间太大。
- 直接切掉头。
- 断手断脚。
- 水平线压在人身上，把人切成两段。
- 镜头歪斜。
- 画面没有主体。
- 产品成了主体，人反而被忽略。

拍个人IP内容时，主体通常是“人”。产品、环境和道具都服务于人，不要抢走注意力。

## 三、运镜

运镜是镜头的移动方式。课程强调：短视频不要为了炫技刻意运镜，运镜要跟着脚本和人物动作自然发生。

常见运镜：

### 推

镜头向前靠近。

适合：

- 开头进入主题。
- 强调重点。
- 让用户更靠近人物。

### 拉

镜头向后远离。

适合：

- 结尾收束。
- 展示环境。
- 从细节回到整体。

### 横摇/摇镜

镜头横向移动或扫过环境。

适合：

- 交代空间。
- 展示两个说话对象。
- 展示店面、工厂、区域。

### 跟随

镜头跟着人物移动。

适合：

- 过程视频。
- 带用户参观。
- 跟着IP走进某个空间。

### 环绕

围绕主体拍摄。

适合：

- 展示人物或产品状态。
- 制造更丰富的空间感。

## 运镜原则

### 1. 运镜跟着脚本走

脚本里有动作，运镜自然出现。

例如：

```text
“跟我走，带你看一下生产区。”
-> 镜头跟随人物走。

“这边是我们的产品部。”
-> 镜头摇到产品部。

“镜头回来，看我手上这个样品。”
-> 镜头从环境回到人物或产品。
```

### 2. 不要刻意炫技

短视频的运镜不需要像电影一样复杂。真实、自然、能帮助用户理解内容就够了。

### 3. 过程类视频可以稍微不稳

课程提到，短视频里适度的不稳定反而有真实记录感，不必追求完全电影化的平稳。

## 拍摄检查表

```text
1. 这条视频适合口播还是过程？
2. 口播用自拍、偷拍、聊天还是采访？
3. 过程视频有没有拍完整流程？
4. 当前镜头是什么景别？
5. 这个景别在表达环境、动作、情绪还是细节？
6. 主体是不是明确？
7. 构图有没有顶头、断手、断脚、没留白？
8. 运镜是不是跟脚本动作自然发生？
9. 画面是不是在突出人设和IP？
```

## 行动清单

- 先把账号主呈现形式定为口播或过程。
- 同一条口播可以换视角重拍，测试流量变化。
- 每条过程视频都先拆流程，再拍素材。
- 口播多用中景、近景，少用远景。
- 构图优先保证主体明确。
- 练熟中心构图、三分构图、前景式构图。
- 运镜跟着脚本动作走，不刻意炫技。
