## 一句话结论

持续更新不是靠每天等灵感，而是靠提前搭建素材库。素材库要分成选题、开篇、金句和呈现形式四类，既收同行，也收跨行业可迁移结构；真正做内容时，把素材拆成元素，再结合自己的赛道、人设和产品重新组合。

## 为什么要搭素材库

课程开头强调：很多博主能多年不断更，不是因为每天都有天降灵感，而是因为他们一直在收集、整理和复用素材。

创作来源主要有两类：

- 之前见过的东西。
- 正在经历的事情。

如果平时没有积累，真正要拍时就容易卡住。

课程用做饭类比：

```text
招待客人前，要先准备食材。
做内容前，也要先准备素材。
```

粉丝就像客人，你不能只做自己想做的菜，而要提前知道他们爱吃什么、关心什么、愿意看什么。

## 创作不是凭空发明

课程提醒：

```text
抖音上没有那么多全新的东西，爆款往往是旧元素的新排列。
```

一个看起来新鲜的视频，背后可能是多个旧视频、旧结构、旧情绪点的重组。

所以素材库的价值不是让你照搬，而是让你积累可拆解的元素：

- 选题。
- 开篇。
- 冲突。
- 结构。
- 情绪。
- 金句。
- 画面形式。
- 拍摄方法。

## 普通人和专业团队的差别

普通人常见流程：

```text
今天想拍
-> 现想选题
-> 想不出来
-> 断更
```

专业团队流程：

```text
先搭素材库
-> 从素材库里找选题
-> 换成自己的赛道
-> 套结构
-> 写脚本
-> 拍摄发布
```

差别不在于专业团队每天更聪明，而在于他们提前准备。

## 素材库四大类

课程明确提到素材库可以分成四类：

```text
选题素材库
开篇素材库
金句素材库
呈现形式素材库
```

这四类解决的是一条视频最核心的四个问题：

- 拍什么。
- 怎么开头让人停下。
- 怎么让用户记住或转粉。
- 怎么拍得更有新鲜感。

## 1. 选题素材库

选题解决“这条视频讲什么”。

### 同行选题

可以收集同行的高赞、高互动、高转化选题。

注意：不要只盯头部大号。

原因：

- 大号有粉丝和IP势能。
- 大号随便拍日常也可能有数据。
- 对新账号参考价值有限。

更适合参考：

- 粉丝量没有大到离谱的账号。
- 半年内发布的视频。
- 点赞明显高于账号平均水平的视频。
- 爆发不是完全靠人设，而是选题本身有吸引力的视频。

### 跨行业选题

课程强调，不要只看本行业。

比如别的行业讲“爱情和前途哪个重要”，你的赛道也可以改成：

- 带货和人设哪个重要。
- 流量和转化哪个重要。
- 选题和拍摄哪个重要。
- 粉丝和成交哪个重要。

真正可迁移的是底层矛盾和用户选择，而不是原句。

### 选题判断标准

可收录的选题最好满足：

- 近期有效，不是很多年前的旧热点。
- 有明显爆发，比账号平时数据高。
- 自己账号能执行。
- 不需要过高拍摄成本。
- 可以换成自己的行业关键词。
- 能回答目标用户真实关心的问题。

## 2. 开篇素材库

开篇解决“用户为什么停下来”。

好的开篇可以从别的行业借结构，再换成自己的赛道。

例子结构：

```text
信不信，10秒让你放弃[某个错误选择]。
```

可改成：

```text
信不信，10秒让你放弃盲目囤货。
信不信，10秒让你放弃只拍产品不做人设。
信不信，10秒让你放弃只靠低价带货。
```

课程提到：如果只看同行，大家互相参考，内容会很像；如果看跨行业的开篇，反而更容易做出新鲜感。

## 小红书也可以找开篇

课程提醒，小红书的标题和封面文案很适合参考开篇。

原因：

- 小红书更依赖标题和封面吸引点击。
- 许多标题一句话就能讲清角度。
- 适合提炼成短视频开头。

做法：

- 搜行业关键词。
- 看高互动笔记标题。
- 提取吸引人的表达。
- 换成自己的视频开篇。

如果自己只想到关键词，可以先去小红书搜，看别人如何把关键词包装成更有点击欲的标题。

## 3. 金句素材库

金句解决“用户为什么记住你、关注你”。

课程提醒：金句不是每种视频都必须强行加。

适合加金句的内容：

- 讲故事。
- 聊观点。
- 晒过程。
- 情绪类内容。
- 人设类内容。

不一定适合强行加金句的内容：

- 很纯的教知识。
- 工具步骤教程。
- 已经讲得很实用的干货。

金句要和前文情绪一致，不能突然拔高，也不能和前面内容没有关系。

好的金句作用：

- 总结情绪。
- 提升格局。
- 让用户产生共鸣。
- 增加转粉概率。

## 金句不要硬套

课程中特别提醒：很多人把金句理解成随便找一句百度名言或名人名言塞进去，这不是好金句。

好金句应该：

- 和前文内容有关。
- 能承接故事或观点。
- 不空泛。
- 不突兀。
- 能让用户觉得“说到我心里了”。

差的金句：

```text
随便百度一句励志名言，和前面内容没关系。
```

好的金句：

```text
前面所有情绪都指向这句话，说出来时用户刚好被击中。
```

## 4. 呈现形式素材库

呈现形式解决“同样的内容怎么拍得不一样”。

课程强调：新的呈现形式不是一定要高制作、高成本、高剪辑，而是给用户新的画面感。

可以改变：

- 场景。
- 服装。
- 道具。
- 镜头位置。
- 说话状态。
- 动作。
- 背景。
- 拍摄对象。
- 拍摄流程。

同一个观点，换一个场景、动作和画面关系，用户就会觉得新鲜。

## 跨行业借呈现形式

不要只看同行怎么拍。

美食、情感、家居、装修、知识、剧情、采访等领域，都可能有可借鉴的呈现方式。

例如：

- 一个美食账号的剪辑节奏，可以借给产品展示。
- 一个装修账号的过程记录，可以借给创业过程。
- 一个采访类账号的提问方式，可以借给人设内容。
- 一个情感号的情绪推进，可以借给观点视频。

关键是看它“怎么呈现”，不是照抄它“讲什么”。

## 素材库来源

可以从以下地方收集：

- 抖音搜索。
- 抖音收藏夹。
- 同行账号。
- 跨行业账号。
- 小红书标题和封面。
- 用户评论区。
- 自己正在经历的事情。
- 自己粉丝关注的问题。

评论区尤其重要。

很多用户在评论里提出的问题，原作者不一定回答。你可以把这些问题变成自己的选题。

## 抖音搜索怎么用

操作思路：

```text
搜索行业关键词
-> 看视频结果
-> 关注近半年内容
-> 找点赞高于账号平均水平的视频
-> 判断自己能不能执行
-> 拆选题/开篇/结构/金句/呈现
```

不要只看播放量或粉丝量，要看这条内容相对于账号平时表现是否突出。

## 素材库怎么分类

电脑上可以建文件夹：

```text
素材库
├── 选题
│   ├── 观点
│   ├── 教知识
│   ├── 晒过程
│   └── 讲故事
├── 开篇
├── 金句
└── 呈现形式
```

手机上可以用：

- 抖音收藏夹。
- 备忘录。
- 笔记软件。
- 表格。

抖音收藏夹建议按用途命名：

- 选题。
- 开篇。
- 金句。
- 呈现形式。
- 观点。
- 故事。
- 拍摄参考。
- 同行参考。

## 收集时记录什么

每条素材最好记录：

- 原视频链接或账号。
- 属于哪类素材。
- 原始标题/开篇。
- 为什么吸引人。
- 可迁移结构。
- 适合改到哪个赛道。
- 自己的改写方向。
- 风险点：是否太像原作、是否不适合自己人设。

## 素材不是照抄

课程的底层意思是：素材库用于拆解和重组，不是复制。

正确方式：

```text
拆出结构
-> 换成自己的赛道
-> 加入自己的经历
-> 用自己的语言表达
-> 换自己的画面呈现
```

错误方式：

```text
原话照搬
镜头照搬
故事照搬
人设照搬
```

个人IP带货要避免让用户觉得“你在抄别人”，要把素材转化成自己的经验和表达。

## 素材库使用流程

```text
1. 今天要发什么内容类型？
2. 先去对应素材库找选题。
3. 再去开篇库找停留方式。
4. 根据内容类型决定要不要加金句。
5. 去呈现形式库找拍法。
6. 改成自己的行业、产品、人设和经历。
7. 写脚本并拍摄。
8. 发布后把数据好的内容反向收进素材库。
```

## 个人IP带货账号怎么用

个人IP带货账号尤其需要素材库，因为既要做人设，又要带产品。

建议重点积累：

- 用户痛点选题。
- 产品使用场景。
- 真实体验开篇。
- 反差开篇。
- 信任背书金句。
- 选品标准金句。
- 产品前后对比呈现。
- 晒过程呈现。
- 评论区问题。
- 成交反馈和用户案例。

## 素材库检查表

```text
1. 是否按选题、开篇、金句、呈现形式分类？
2. 是否只收藏同行，缺少跨行业素材？
3. 是否只收藏链接，没有记录可迁移结构？
4. 是否关注近半年有效内容？
5. 是否避开执行成本过高的素材？
6. 是否把评论区问题变成选题？
7. 是否定期清理过时素材？
8. 是否把自己跑得好的视频反向沉淀？
```

## 行动清单

- 今天先建四个基础文件夹：选题、开篇、金句、呈现形式。
- 每天刷到好内容，至少拆一条进素材库。
- 每周从同行收集10条，从跨行业收集10条。
- 每条素材都写一句“我能怎么改到自己的赛道”。
- 每周从评论区整理20个用户问题。
- 每次写脚本前，先从素材库组合：选题 + 开篇 + 呈现形式。
- 每月复盘素材库，把过时、不可执行、太像原作的内容清掉。
