## 一句话结论

DOU+本质是抖音的付费流量工具，不是用来拯救所有烂视频的，而是用来放大已经有潜力的内容、补足面子数据、获取精准粉丝、推动商品成交或放大直播间流量。投放前要先明确目标，投放后要看成本、转化和ROI，跑不动就停，不要盲目追投。

## DOU+是什么

课程把 DOU+ 类比成线下发传单：

```text
你花钱，让抖音把你的视频推给更多人。
```

常规理解：

- DOU+是付费流量。
- 大致可以理解为用钱购买额外曝光。
- 课程中提到常见基础量级是 100 元约 5000 播放，但实际会受投放目标、人群定向、平台规则等影响。

不要把 DOU+理解为万能按钮。它只是放大器，视频本身没有吸引力，付费放大后也可能只是更快暴露问题。

## 投放入口

个人账号常见路径：

```text
打开视频
-> 点击右侧/下方更多
-> 上热门 / DOU+
-> 选择投放目标
-> 设置预算、时长、人群
-> 支付并等待投放结果
```

不同账号类型入口略有差异：

- 个人号：主要从单条视频的更多入口进入。
- 企业号：可能有获取客户、线索、商品推广等更多目标。
- 蓝V或本地生活账号：可能使用本地推，围绕门店流量、团购成交、线下互动等目标投放。

## 投放目标

DOU+不是只有“买播放”。不同投放目标会匹配不同人群。

### 1. 内容加热

常见目标：

- 播放量。
- 点赞评论。
- 主页访问。

适合：

- 想让内容被更多人看到。
- 想补足互动数据。
- 想让用户进入主页进一步了解你。

### 2. 粉丝经营

常见目标：

- 涨粉。
- 给粉丝推送。
- 让已有粉丝看到某条重要内容。

适合：

- 课程、活动、直播、上新等需要通知老粉的内容。
- 想积累账号未来商业价值的内容。

课程举例：如果一个老师开了新的训练营，最好优先推给认识、信任、认可他的粉丝，而不是直接推给完全陌生的人。

### 3. 商品推荐

常见目标：

- 商品曝光。
- 商品点击。
- 商品购买。
- ROI目标。

适合：

- 挂小黄车视频。
- 产品种草视频。
- 短视频带货。

重点不是单纯播放，而是点击、下单和付款。

### 4. 直播间投放

常见目标：

- 直播间人气。
- 直播间涨粉。
- 直播间打赏。
- 直播间互动。

平台会根据目标把流量推给对应行为偏好的人群。

例如：

- 投直播间互动，会更倾向于找喜欢在直播间评论互动的人。
- 投直播间打赏，会更倾向于找有打赏行为的人。

### 5. 企业号和本地推

企业号可能会多出：

- 抖音私信线索。
- 评论链接线索。
- 落地页线索。
- 联系电话。
- 门店流量。
- 团购成交。

本地生活账号尤其适合设置地域和门店目标，否则流量可能花到不可能到店的人身上。

## 智能推荐和自定义人群

投放时通常会遇到两类人群选择：

- 系统智能推荐。
- 自定义人群。

自定义人群可以按以下维度设置：

- 性别。
- 年龄。
- 地域。
- 兴趣分类。
- 行为偏好。

选择越精准，流量可能越贵，因为平台需要更精细地找人。

判断逻辑：

```text
目标人群非常明确，就用自定义。
目标人群不够明确，先用智能推荐测试。
```

个人IP带货账号如果知道自己的核心购买人群，比如宝妈、健身人群、本地用户、某年龄段女性，就可以适当自定义，减少无效曝光。

## 投放时长和预算

常见投放时长包括：

- 2小时。
- 12小时。
- 24小时。

课程给出的思路：

- 如果视频面子数据还过得去，可以小预算拉长到 24 小时。
- 如果刚发布时点赞评论太少，担心用户看到后不愿互动，可以先投 2 小时快速补数据。
- 2小时跑完后，如果数据好，再追加 24 小时。

简单策略：

```text
先小额测试，再根据结果追加。
```

不要一开始就大额砸钱，尤其是还没有验证视频质量和转化时。

## 投什么：现在钱、未来钱、面子

课程把 DOU+投放目标分成三类。

### 1. 投现在的钱：商品成交

目标是让挂车视频获得更多商品曝光、点击和购买。

看的是：

- 商品点击率。
- 下单数量。
- 支付数量。
- ROI。

适合已经有明确产品和成交路径的视频。

### 2. 投未来的钱：粉丝量

目标是积累粉丝，提升账号未来商业价值。

粉丝多了，未来接广告、卖课、带货、直播、私域转化都会更有基础。

但投粉丝要算成本。

```text
单粉成本 = 投放金额 / 新增粉丝数
```

如果 100 元涨 425 个粉丝，约 0.24 元/粉，需要看这个成本是否符合自己的账号阶段。

如果 30 元只涨 4 个粉丝，单粉成本约 7.5 元，通常就不适合继续投。

### 3. 投面子：点赞评论

目标是让视频看起来更有信任感。

课程举例：两条同样选题的视频，一条点赞很少，一条点赞很多，用户天然更容易认为后者更值得看。

面子数据的价值：

- 降低用户划走概率。
- 增强从众心理。
- 提升内容可信度。
- 让新用户更愿意互动。

适合：

- 内容本身不错，但初始互动弱。
- 视频会被用于主页沉淀、广告背书或转化入口。

## 面子工程怎么投

两种策略：

### 1. 数据还可以

如果视频自然反馈还可以，只是想让互动更好看：

```text
小预算 + 24小时 + 点赞评论
```

让系统慢慢跑，不急着短时间冲量。

### 2. 刚发布需要快速补初始数据

如果刚发布时点赞评论太少，担心影响用户第一印象：

```text
先投2小时
-> 看结果
-> 数据不错再追加24小时
```

这种更像给视频先打底，再决定是否放大。

## 粉丝量怎么投

投涨粉要看两个结果：

- 粉丝画像准不准。
- 单粉成本划不划算。

复盘维度：

- 新增粉丝数量。
- 男女性别比例。
- 年龄段分布。
- 地域分布。
- 是否符合账号目标人群。
- 单粉成本。

如果投来的粉丝和你的目标人群差很多，就算便宜也未必值得。

个人IP带货账号不要只追求粉丝数字，更要看粉丝有没有未来购买、咨询、复购或传播价值。

## 商品推广怎么投

商品推广可以围绕：

- 商品购买。
- 商品曝光。
- ROI目标。
- 自动出价。
- 手动出价。
- 自定义人群。

核心不是“花了多少钱”，而是投放漏斗是否成立。

漏斗可以这样看：

```text
曝光
-> 点击小黄车
-> 下单
-> 付款
-> 产出金额
```

如果花 100 元，只卖出 70 元销售额，而且还没扣成本和佣金，继续投通常不划算。

如果花 100 元，带来明显高于成本的成交，才考虑追加。

## ROI判断

ROI可以理解为投入产出比：

```text
ROI = 成交金额 / 投放金额
```

但实际判断不能只看成交金额，还要扣除：

- 产品成本。
- 平台扣点。
- 佣金。
- 退货。
- 人工和素材成本。

课程中的判断原则很直接：

```text
亏了就停，赚了再追。
```

## 视频不适合投放怎么办

有些视频会提示不适合投放或审核不过。

课程给出的处理路径：

```text
创作者/商家相关入口
-> 投放管理
-> 客服
-> 输入“人工服务”
-> 选择审核问题
-> 发视频链接
-> 询问不通过原因
```

客服可能会提示：

- 营销感过强。
- 表达违规。
- 内容不适合投放。
- 商品或画面存在风险。

根据反馈修改后，再尝试重新投放。

## 投放中途后悔怎么办

如果订单已经开始消耗，但不想继续投，可以尝试终止订单。

路径同样通过客服或订单管理：

```text
投放管理
-> 客服/订单
-> 终止订单
```

未消耗的余额通常会退回 DOU+账户，后续可以继续使用。

## DOU+不是每条视频都要投

课程提到，成熟账号并不一定大量投 DOU+。更重要的是让多条视频同时自然跑。

核心思路：

```text
不要把所有注意力压在单条视频上。
```

每天只盯一条视频，很容易焦虑。持续发布多条内容，让多个视频同时跑，才能降低对单条数据的执念，也更容易遇到自然爆发。

## 什么视频值得特殊投

虽然不是每条都投，但有些视频值得重点放大：

- 能直接带来成交的视频。
- 能强力立人设的视频。
- 能让用户产生信任的视频。
- 评论区有大量咨询的视频。
- 自然数据已经证明有潜力的视频。

判断标准：

```text
它能不能帮我成交、立人设、涨精准粉或建立信任？
```

如果答案明确，再考虑追加预算。

## 为什么越投效果可能越差

投放不是无限复制同样结果。

课程解释：目标人群池是有限的。

前期投放时，平台先找到最精准、最容易响应的人。随着不断追加，精准人群被逐渐消耗，后面可能会扩展到不那么精准的人群，所以效果下滑。

常见表现：

- 100 元第一次涨粉很多。
- 继续追加后，同样 100 元涨粉明显减少。
- 粉丝画像开始变散。
- 单粉成本变高。
- 转化变差。

处理方式：

- 一旦边际效果明显下滑，就停止。
- 不要期待同一条视频无限放大。
- 继续生产新内容，寻找新的可放大素材。

## 资产和负债的视角

课程最后把短视频流量放在商业资产里理解。

商业可以分为三个环节：

- 产品端。
- 流量端。
- 交付端。

如果没有极强产品、极强品牌、极强技术专利或极强交付能力，普通人最容易切入的往往是流量端。

短视频账号的价值在于：

- 持续获得用户注意力。
- 把用户沉淀到账号、直播间、私域或成交路径。
- 用内容放大生意，而不是只依赖线下门店、工厂、固定成本。

课程想表达的是：

```text
流量能力可以把生意从负债压力转成可放大的资产。
```

## 个人IP带货的DOU+投放原则

```text
1. 先明确目标，再决定投放选项。
2. 不要用DOU+拯救烂内容。
3. 小预算测试，数据好再追加。
4. 普通内容看停留和互动，带货内容看点击和成交。
5. 投涨粉要看画像和单粉成本。
6. 投商品要看ROI，不赚钱就停。
7. 投点赞评论是为了提升信任感，不是自我安慰。
8. 越投越差时及时停，不硬追。
9. 真正值得大投的是能成交、立信任、立人设的视频。
10. 持续生产内容，比纠结单条视频更重要。
```

## 行动清单

- 给每条待投视频先标注目标：播放、互动、涨粉、成交、直播、门店。
- 先用小预算测试 2小时或24小时效果。
- 每次投放记录预算、时长、目标、人群、结果。
- 投粉丝记录单粉成本和粉丝画像。
- 投商品记录曝光、点击、下单、付款和ROI。
- 投放效果下滑时停止，不继续硬砸。
- 每周复盘哪些视频值得作为“可放大资产”。
