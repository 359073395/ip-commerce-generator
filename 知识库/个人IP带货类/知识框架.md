# 个人IP带货知识框架

## 总链路

```text
定位 -> 人设 -> 内容 -> 爆款 -> 直播 -> 私域 -> 成交 -> 交付 -> 复购 -> 全域增长
```

## 1. 定位与人设

解决问题：

- 用户为什么要记住你。
- 你代表什么身份、观点和结果。
- 你和同行的差异是什么。

对应课程：

- V001/V013 黄予：艺术文化 IP。
- V020/V003/V004 玺颜：素人 IP 起盘和陪跑。
- V014 参哥：操盘手与流量赋能认知。

## 2. 内容获流

解决问题：

- 用户为什么停留。
- 用户为什么点赞、评论、转发。
- 泛内容和垂直内容如何配比。

对应课程：

- V005/V006 刘佳康：爆款文案。
- V008 青霞：爆款拍摄。
- V015 杰森：短视频趋势。
- V020 玺颜：素人 IP 起盘内容比例。

## 3. 直播承接

解决问题：

- 内容流量如何进入直播间。
- 直播间如何留人、互动、转化。
- 货盘、话术和复盘如何设计。

对应课程：

- V007/V009 美晞：直播运营核心维度。
- V012 志浩：私域直播。
- V020 玺颜：直播引流。

## 4. 私域成交

解决问题：

- 流量如何沉淀。
- 朋友圈、社群、销售 SOP 如何配合。
- 一转、二转、复购、售后如何跑通。

对应课程：

- V018 真真：私域全流程。
- V012 志浩：私域直播成交。
- V002 冷哥：服务型 IP 私域转化。

## 5. 产品与服务

解决问题：

- 个人 IP 最终卖什么。
- 产品、课程、服务、门店如何承接。
- 如何用标准化服务支撑规模化。

对应课程：

- V016/V017 老纪：实体店 IP 带货与服务标准化。
- V019 张心媛：全域增长和二转三转。
- V002 冷哥：设计服务转化。

## 6. AI与规模化

解决问题：

- 如何提高内容生产效率。
- 如何让个人 IP 有稳定内容供给。
- 如何搭建自动化生产线。

对应课程：

- V010/V011 鲜辉：AI 短视频 IP 自动化生产线。

## 一句话总纲

个人IP带货不是“人设”和“卖货”的简单相加，而是用 IP 获得信任，用内容获得流量，用直播和私域承接需求，再用产品、服务和复购完成商业闭环。
